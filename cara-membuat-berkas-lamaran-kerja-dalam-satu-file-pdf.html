Jawaban:

Cara menulis surat lamaran kerja :

1. Tempat dan tanggal penulisan surat.

2. Kepada Yth. nama orang beserta gelar dan jabatannya atau perusahaan yang dituju.

3. Alamat lengkap perusahaan juga tidak boleh dilewatkan.

4. Salam pembuka

5. Kata pengantar/pembukaan

Saat menuliskan kata pengantar atau pembuka dalam surat lamaran kerja, cantumkan di mana Anda mendapatkan informasi mengenai lowongan kerja yang dibuka pada perusahaan yang Anda tuju. Baik dari surat kabar maupun internet, atau teman Anda.

Sebutkan posisi jabatan pekerjaan yang Anda pilih jika dalam perusahaan tersebut membutuhkan banyak sekali karyawan baru.

6. Biodata pribadi

Tuliskan nama lengkap Anda, tempat dan tanggal lahir, alamat lengkap, nomor telepon atau nomor handphone, E-Mail, Pendidikan (tidak wajib)

Jika Anda ingin menulis latar belakang pendidikan Anda, tulis saja latar belakang pendidikan tertinggi Anda.

7. Pengalaman kerja dan kemampuan Anda

Jika sebelumnya Anda pernah bekerja, tulisan pekerjaan Anda sebelumnya. Jika belum pernah bekerja (fresh-graduate) tidak usah. Tulis juga kemampuan Anda yang sekiranya dibutuhkan dalam perusahaan tersebut.

8. Penutup

Tulislah bahwa Anda mempunyai keinginan besar agar diterima di perusahaan tersebut. Jangan lupa mengucapkan Terima kasih.

9. Lampiran surat lamaran kerja

Agar perusahaan yang Anda tuju semakin yakin untuk menerima Anda, lampirkan juga hal-hal yang dapat menambah nilai Anda. Hal yang sebaiknya ada sebagai lampiran adalah:

Pas foto (foto terbaru)

Fotokopi KTP

Daftar riwayat hidup

Fotokopi ijazah terakhir

Fotokopi sertifikat kompetensi (kursus atau pelatihan yang pernah Anda ikuti)

PEMBAHASAN :

Sebelum menjawab soal tersebut, mari kita mengenal terlebih dahulu file yang berekstensi pdf ^_^

→ Portable Document Format atau biasa disingkat dengan PDF, merupakan suatu ekstensi yang dikembangkan oleh perusahaan Adobe Inc pada tahun 1993.

→ Ektensi PDF ini memungkinkan user untuk dapat merubah baik file berupa dokumen (doc atau docx) dan gambar menjadi satu file berekstensi PDF yang praktis dengan kompabilitas tinggi.

→ Untuk dapat membuka file PDF, diperlukan aplikasi reader khusus seperti Adobe reader, Foxit PDF, Nitro PDF, dan banyak aplikasi pihak ke 3 lainnya.

→ Saat ini tingkat penggunaan file berekstensi PDF sangat tinggi, disebabkan oleh beberapa kelebihan yang dimilikinya, seperti :

membutuhkan ruang penyimpanan (memori) yang relatif kecil dibanding ekstensi lainnyafile dokumen berupa tulisan atau grafik akan dianggap sebagai gambar, sehingga memudahkan untuk perpindahan halaman saat membaca ebooktidak adanya kekhawatiran terhadap kemungkinan salah tekan atau perubahan konten ebook yang tidak disengaja, karena file pdf akan menganggap isinya sebagai sebuah gambar yang solid, dan hanya bisa di edit saat memiliki aplikasi pdf editor yang rata-rata berbayarlebih tahan akan kemungkinan kerusakan dokumen, di banding format docx

Nah, setelah tau kelebihan file berekstensi PDF, lalu timbul pertanyaan bagaimana merubah file berformat dokumen (doc), menjadi PDF ?

Langkah yang dapat dilakukan adalah sebagai berikut :

Buka file berekstensi dokumen yang akan Anda ubah menjadi file berekstensi PDF (dalam kasus ini penulis menggunakan aplikasi Ms. Word 2013)Buka tab file, dan pilih eksporPilih "Buat PDF/XPS"Lalu beri nama file berekstensi PDF yang Anda inginkanDan klik "terbitkan"Selamat file berekstensi dokumen Anda sudah menjadi file berformat PDF :)

*Jika Anda memiliki aplikasi pihak ke 3, seperti foxit phantom PDF, maka pengubahan file dokumen atau gambar akan menjadi lebih mudah, yakni hanya dengan klik kanan kursor pada file yang akan diubah ekstensinya, lalu pilih "convert to pdf"

26. Aplikasi yang dipakal Tindakan Percobaan Aplikasi pengolah lembar kerja Bukalah sebuah file .docx yang telah disiapkan guru. Simpan sebuah file .docx menjadi .rtf dan buka file .rtf tersebut. Aplikasi pengolah Simpan sebuah file.docx menjadi kata pdf dan buka file .pdf tersebut. Simpan sebuah file .docx menjadi .xlsx, dengan menulis langsung nama ekstensinya saat Save As. Buka file .xlsx tersebut. Buka sebuah file lembar kerja (.xlsx). Simpan sebuah file .xlsx menjadi .txt dan buka kembali file .txt tersebut dengan aplikasi pengolah lembar kerja. Simpan sebuah file .xlsx menjadi .csv dan buka kembali file .csv tersebut dengan aplikasi pengolah lembar kerja. Simpan sebuah file .xlsx menjadi .pdf dan buka kembali file .pdf tersebut dengan aplikasi pengolah lembar kerja. Simpan sebuah file .xlsx menjadi .docx Apa yang terjadi? Apa kesimpulan kalian? Mengapa demikian?​


Percobaan dengan word processor atau pengolah kata:

Jika kita menyimpan sebuah file .docx menjadi .rtf dan membuka file .rtf tersebut, yang terjadi adalah file .docx akan dikonversi ke format .rtf, yang merupakan rich text format.  Jika kita menyimpan sebuah file .docx menjadi .pdf dan membuka file .pdf tersebut, yang terjadi adalah file .docx akan dikonversi ke format .pdf, yang merupakan format dokumen portabel (portable document format). Jika kita menyimpan sebuah file .docx menjadi .xlsx, dengan menulis langsung nama ekstensinya saat Save As. Buka file .xlsx tersebut, yang terjadi adalah file .docx akan disimpan dengan ekstensi .xlsx, tetapi tidak akan dikonversi ke format .xlsx, yang merupakan format spreadsheet (lembar kerja).

Pembahasan

Ada pun bila dilakukan percobaan dengan aplikasi spreadsheet atau pengolah lembar kerja:

Jika kita membuka sebuah file lembar kerja (.xlsx) dengan aplikasi pengolah lembar kerja, yang terjadi adalah file .xlsx akan terbuka dengan normal, karena formatnya sesuai. Jika kita menyimpan sebuah file .xlsx menjadi .txt dan membuka kembali file .txt tersebut dengan aplikasi pengolah lembar kerja, yang terjadi adalah file .xlsx akan dikonversi ke format .txt, yang merupakan format teks biasa (plain text). Jika kita simpan sebuah file .xlsx menjadi .csv dan membuka kembali file .csv tersebut dengan aplikasi pengolah lembar kerja, yang terjadi adalah file .xlsx akan dikonversi ke format .csv, yang merupakan format comma separated values.
