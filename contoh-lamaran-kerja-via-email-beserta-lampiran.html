10. Cara untuk menyertakan lampiran pada email yang kita buat


1 siapkan terlebih dahulu file-file yang akan digunakan sebagai lampiran. Tulis isi surat, lalu klik tombol Lampirkan File, selanjutnya ditampilkan halaman untuk memasukkan lampiran-lampiran seperti gambar di bawah ini.

2Klik tombol Browse untuk mencari file yang akan disertakan, kemudian akan ditampilkan kotak dialog Choose file seperti gambar di bawah ini.

3Klik file yang dimaksud, selanjutnya klik tombol Open. Perhatikan gambar di bawah ini, file lampiran sudah masuk di kotak sisipan pertama.
Jika file yang ingin dilampirkan lebih dari satu, maka dapat kamu lakukan dengan langkah yang sama. Selanjutnya e-mail siap untuk dikirim dengan mengklik tombol Kirim.

15. Contoh yang tidak dapat dilampirkan pd email


Semua jenis file umumnya bisa dikirim via email.
Jika file yang akan dikirim berisi virus, maka biasanya akan discan secara otomatis.
Namun batas maximal ukuran file dibatasi maksimal 25 mb (Gmail, Yahoo & Outlook).[jawaban]

Mata Pelajaran: Teknologi Informatika
Kata kunci: Lampiran Email

• Jawaban pendek:

Contoh yang tidak dapat dilampirkan pd email

Untuk melampirkan file pada Email, file yang akan dilampirkan dapat berupa gambar, audio, video, dan file dalam berbagai format yang belum dikompresi (Word, Excel, PowerPoint, PDF, JPG, dll) ataupun format yang telah dikompresi seperti ZIP atau RAR. 

Yang tidak bisa dilampirkan pada file Email adalah file dengan format .EXE

23. fasilitas yang dapat menyertai lampiran pada email disebut​


attachment

Penjelasan :

Attachment berguna untuk melampirkan file pada email. Dengan adanya fasilitas attachment, pengguna email bukan hanya dapat mengirim pesan, namun juga dapat mengirim file kepada pengguna email lainnya.

File-file yang dapat dilampirkan dalam fasilitas attachment antara lain: file dokumen, file gambar serta file audio.
