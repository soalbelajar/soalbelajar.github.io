19. fungsi file excel dan word pada mailing


Jawaban ringkas:

File Excel berfungsi sebagai sumber data (data source) dalam pengisian Mailing.

File Word berfungsi sebagai dokumen data utama dalam penulisan Mailing.

Penjelasan panjang:

Mailing adalah salah 1 fasilitas di Ms. Word untuk menulis surat massal. Artinya yaitu: surat yg formatnya sama, namun tujuan, alamat dan nama orang yg dikirimi surat berlainan (biasanya lebih dari 5 orang).

Pengetikan format surat dilakukan di Ms. Word. Sementara sumber datanya (contoh: tujuan, alamat, nama orang) dapat dibuat di Ms. Excel.

Misalkan di program Ms. Word 2013, amatilah lampiran:

- Mailings ditunjukkan tanda panah biru

- Select Recipients ditunjukkan tanda panah merah

- Use an Existing List ditunjukkan tanda panah hijau

- Akan tampil Select Data Source (ditunjukkan tanda panah coklat)

- Di bagian yg ditunjukkan tanda panah hitam, pilih (klik) Excel Files

- Langkah terakhir ialah melakukan dobel klik file Excel yg sudah dibuat

40. "apa perbedaan microsoft word dan microsoft excel?"


Perbedaan Microsoft Word dan Microsoft Excel adalah :

Microsoft word berfungsi untuk mengolah kata dimana didalamnya terdapat lembar kerja yang berupa blank pages dengan segala fiturnyaMicrosoft excel berfungsi untuk mengolah data dan angka dimana didalamnya terdapat lembar kerja yang berwujud spreadsheet dan dapat digunakan operator fungsi matematis di dalamnya.Pembahasan

Perbedaan microsoft word dan excel sangat khas karena keduanya memiliki fungsi yang berbeda dalam membantu penggunanya.

Microsoft word akan lebih cocok digunakan pengguna yang bertujuan untuk mengerjakan dokumen yang mengandung banyak kata - kata seperti makalah, laporan observasi dan sebagainya. Fitur didalamnya juga mendukung dalam pengerjaan dokumen seperti pengaturan pages berupa cover, ukuran, orientasi, margin, page colour, page number, mengatur tata letak lampiran dan masih banyak lagi. Hal - hal seperti ini tidak dapat ditemui di dalam Microsoft Excel. Microsoft word juga menawarkan fitur membuat tabel dan chart akan tetapi, chart biasanya akan terhubung dengan microsoft excel karena microsoft word tidak dapat mengolah angka menggunakan fungsi - fungsi matematis. Ekstensi file pun berbeda, microsoft word akan menggunakan ekstensi doc atau docx. Lembar kerja yang disediakan memiliki wujud yang berbeda, pada microsoft word, tampilan lembar kerja akan berupa sebuah blank page yang sudah terdapat margin yang tersetting secara default dan dapat diubah.

Lain halnya dengan Microsoft Excel, software ini lebih cocok bagi mereka yang mengolah data berupa angka karena didalamnya memungkinan melakukan operasi matematis dan banyak fungsi lain contohnya =SUM, =AVERAGE, = MAX, =COUNT , dan lain - lain.  Tampilan excel juga berupa spreadsheet dan memiliki nomor baris beserta nama kolom menggunakan alphabet.
