jumlah deret = (2011*(2011+1)(22011+1))/6

jumlah deret = (201120124022)/6

jumlah deret = 48,532,947,872

11. Berapa nilai jumlah 10 suku dari deret kuadrat bilangan asli


Nilai jumlah 10 suku dari deret kuadrat bilangan asli adalah 385.

Pembahasan

Pola bilangan adalah kumpulan bilangan yang diurutkan berdasarkan aturan tertentu. Salah satu contoh pola bilangan adalah pola bilangan kuadrat.

Penjumlahan suku-suku terurut dari suatu pola bilangan disebut deret.

Himpunan bilangan asli = A = {1, 2, 3, 4, ...}

Kuadrat bilangan asli = 1², 2², 3², ... => Un = n²

Untuk mencari nilai jumlah 10 suku dari deret kuadrat bilangan asli dapat dilakukan dengan 2 cara :

a) menghitung nilai masing-masing suku lalu dijumlahkan

S₁₀ = U₁ + U₂ + U₃ + U₄ + U₅ + U₆ + U₇ + U₈ + U₉ + U₁₀

S₁₀ = 1² + 2² + 3² + 4² + 5² + 6² + 7² + 8² + 9² + 10²

S₁₀ = 1 + 4 + 9 + 16 + 25 + 36 + 49 + 64 + 81 + 100

S₁₀ = 385

b) Menggunakan Rumus

[tex]S_n=\frac{n(n+1)(2n+1)}{6}[/tex]

[tex]S_{10}=\frac{10(10+1)(2(10)+1)}{6}[/tex]

[tex]S_{10}=\frac{10(11)(21)}{6}[/tex]

[tex]S_{10}=\frac{2.310}{6}[/tex]

[tex]S_{10}=385[/tex]

Jadi nilai jumlah 10 suku dari deret kuadrat bilangan asli adalah 385.

19. deret bilangan asli 1+2+3+........+(n-1)+n. Jumlah 200 suku pertama deret bilangan asli adalah


Deret bilangan : 1 + 2 + 3 + .... + (n - 1) + n. Maka jumlah 200 suku pertama (S₂₀₀) adalah 20.100.

Penyelesaian Soal :

Diketahui : Suku pertama (a) = 1

Ditanya : jumlah 200 suku pertama (S₂₀₀) ?

Jawab :

LANGKAH PERTAMA (I)

Hitung nilai beda (b) dengan menggunakan cara sebagai berikut :

b = Uₙ - Uₙ₋₁

Keterangan : b = beda  

                      Un = suku ke-n

Maka perhitungannya yaitu :

b = U₂ - U₁

  = 2 - 1

  = 1

LANGKAH KEDUA (II)

Hitung jumlah 200 suku pertama (S₂₀₀) dengan menggunakan cara sebagai berikut :

Sₙ = [tex]\frac{n}{2}[/tex] (2a + (n - 1) b)

Keterangan : Sₙ = Jumlah suku n pertama

                       Uₙ = suku ke-n

                       a = suku pertama

                       b = beda

                       n = banyaknya suku

Maka perhitungannya yaitu :

Sₙ = [tex]\frac{n}{2}[/tex] (2a + (n - 1) b)

S₂₀₀ = [tex]\frac{200}{2}[/tex] (2 (1) + (200 - 1) 1)

S₂₀₀ = 100 (2 + 199)

S₂₀₀ = 100 (201)

S₂₀₀ = 20.100

∴ Kesimpulan jumlah 200 suku pertama (S₂₀₀) deret tersebut adalah 20.100.

pembahasan :
