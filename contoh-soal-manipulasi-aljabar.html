a. Lim (x² - 12x + 27)/(3x - 9)

x → 3

= (x - 9)(x - 3)/3(x - 3)

= (x - 9)/3

= (3 - 9)/3

= -6/3

= -2

b. Lim (b² - 8b + 7)/(2b² - 5b + 3)

b → 1

= (b - 7)(b - 1)/(b - 1)(2b - 3)

= (b - 7)/(2b - 3)

= (1 - 7)/(2 - 3)

= -6/(-1)

= 6

15. Buktikan identitas persamaan boolean berikut, menggunakan manipulasi aljabarA'B + B'C' + AB + B'C = 1​


Jawab dan Penjelasan dengan langkah-langkah:

MANIPULASI ALJABAR BOOLEAN

Pembuktian A'B + B'C' + AB + B'C = 1

A'B + B'C' + AB + B'C = 1

(terapkan hukum komutatif)

A’B + AB + B'C' + B'C = 1

(terapkan hukum komutatif)

BA’ + BA + B'C' + B'C = 1

(terapkan hukum asosiatif)

B(A’ + A) + B’(C’ + C) = 1

(terapkan hukum negasi/komplemen)

B(1) + B’(1) = 1

(terapkan hukum identitas)

B + B’ = 1

(terapkan hukum negasi/komplemen)

1 = 1

(terbukti benar)

∴  Jadi, A'B + B'C' + AB + B'C = 1 terbukti benar.

26. 4. Gunakan Manipulasi Aljabar untuk Mendapatkan Hasil Limit dari soal berikut ! B. Lim (x → 3) (x^2 + 4x + 3) / (x + 3) Tolong segera dijawab ia, ini Sangat Urgen ! Terima Kasih.


Jawaban:

Diketahui:

Lim (x → 3) (x^2 + 4x + 3) / (x + 3)

Ditanya:

Nilai limit dari persamaan di atas ketika x mendekati 3.

Langkah Penyelesaian:

Kita dapat menggunakan manipulasi aljabar untuk menyelesaikan limit tersebut.

Pertama, mari kita coba untuk memfaktorkan persamaan pada pembilang. Perhatikan bahwa x^2 + 4x + 3 dapat difaktorkan menjadi (x + 3)(x + 1).

(x^2 + 4x + 3) / (x + 3) = [(x + 3)(x + 1)] / (x + 3)

Kedua, kita bisa mempecah persamaan menjadi dua pecahan:

[(x + 3)(x + 1)] / (x + 3) = (x + 1)

Ketiga, pada limit, kita bisa langsung menggantikan x dengan angka yang diberikan, yakni 3.

(x + 1) = (3 + 1) = 4

Hasil Akhir:

Limit dari persamaan tersebut ketika x mendekati 3 adalah 4.

[tex]( \: manipulasi \: \: aljabar \: ) \\ \\ \: \: \: \: \: \int x \sqrt{2x + 3} \: dx \\ \\ = \int \frac{1}{2} (2x + 3) \sqrt{2x + 3} \: dx - \int \frac{3}{2} \sqrt{2x + 3} \: dx \\ \\ ( \: metode \: \: substitusi \: ) \\ \\ \text{Misal} \: u = 2x + 3 \\ \\ \frac{du}{dx} = 2 \: \Rightarrow \: du = 2 \: dx \\ \\ \: \: \: \: \: \int x \sqrt{2x + 3} \: dx \\ \\ = \int \frac{1}{2} (2x + 3) \sqrt{2x + 3} \: dx - \int \frac{3}{2} \sqrt{2x + 3} \: dx \\ \\ = \int \frac{1}{4} \: {u}^{ \frac{3}{2} } \: du - \int \frac{3}{4} \: {u}^{ \frac{1}{2} } \: du \\ \\ = \frac{1}{4} \int \: {u}^{ \frac{3}{2} } \: du - \frac{3}{4} \: \int {u}^{ \frac{1}{2} } \: du \\ \\ = \frac{1}{4} \left ( \frac{2}{5} {u}^{ \frac{5}{2} } \right ) - \frac{3}{4} \left ( \frac{2}{3} {u}^{ \frac{3}{2} } \right ) + c \\ \\ = \frac{1}{10} \left ( {u}^{ \frac{5}{2} } \right ) - \frac{1}{2} \left ( {u}^{ \frac{3}{2} } \right ) + c \\ \\ = \frac{1}{10} \left ( {(2x + 3)}^{ \frac{5}{2} } \right ) - \frac{1}{2} \left ( {(2x + 3)}^{ \frac{3}{2} } \right ) + c \\ \\ = \frac{1}{5} (x - 1) (2x + 3)^{ \frac{3}{2}} + c \\ \\[/tex]

40. Dengan melakukan manipulasi aljabar, buktikan bahwa (r - q) / (p - r) = q / p ekuivalen dengan 1 / p + 1 / q = 2 / r !


.
(r - q)/ (p- r) = q/p
p(r-q) = q/(p-r)
pr - pq  = pq - qr
pr + qr = pq + pq
r(p+q) = 2 (pq)
(p + q)/(pq) = 2/r
p/pq + q/pq = 2/r
1/q + 1/p =2/r
1/p+1/q  = 2/r
