10. Jumlah suku ke-100 dari deret bilangan asli adalah….


Jawaban:

5050

Penjelasan:

Rumus jumlah suku ke-n dari deret aritmatika adalah:

Sn = (n/2)(a + L)

Dalam hal ini, kita ingin mencari jumlah suku ke-100 dari deret bilangan asli. Dalam deret bilangan asli, suku pertama (a) adalah 1 dan suku terakhir (L) adalah 100.

Menggunakan rumus di atas, kita dapat menghitung jumlah suku ke-100:

S100 = (100/2)(1 + 100)

S100 = 50(101)

S100 = 5050

Jadi, jumlah suku ke-100 dari deret bilangan asli adalah 5050.

20. deret bilangan asli 1+2+3+........+(n-1)+n. Jumlah 200 suku pertama deret bilangan asli adalah


Deret bilangan : 1 + 2 + 3 + .... + (n - 1) + n. Maka jumlah 200 suku pertama (S₂₀₀) adalah 20.100.

Penyelesaian Soal :

Diketahui : Suku pertama (a) = 1

Ditanya : jumlah 200 suku pertama (S₂₀₀) ?

Jawab :

LANGKAH PERTAMA (I)

Hitung nilai beda (b) dengan menggunakan cara sebagai berikut :

b = Uₙ - Uₙ₋₁

Keterangan : b = beda  

                      Un = suku ke-n

Maka perhitungannya yaitu :

b = U₂ - U₁

  = 2 - 1

  = 1

LANGKAH KEDUA (II)

Hitung jumlah 200 suku pertama (S₂₀₀) dengan menggunakan cara sebagai berikut :

Sₙ = [tex]\frac{n}{2}[/tex] (2a + (n - 1) b)

Keterangan : Sₙ = Jumlah suku n pertama

                       Uₙ = suku ke-n

                       a = suku pertama

                       b = beda

                       n = banyaknya suku

Maka perhitungannya yaitu :

Sₙ = [tex]\frac{n}{2}[/tex] (2a + (n - 1) b)

S₂₀₀ = [tex]\frac{200}{2}[/tex] (2 (1) + (200 - 1) 1)

S₂₀₀ = 100 (2 + 199)

S₂₀₀ = 100 (201)

S₂₀₀ = 20.100

∴ Kesimpulan jumlah 200 suku pertama (S₂₀₀) deret tersebut adalah 20.100.
