Tuliskan salah satu contoh fungsi If yang sudah pernah kamu lakukan dalam praktikum.​


Jawaban:

1. Fungsi IF adalah fungsi logika dalam berbagai bahasa pemrograman dan perangkat lunak seperti Excel yang digunakan untuk melakukan pengujian kondisi. Fungsi ini memungkinkan program atau spreadsheet untuk menjalankan tindakan tertentu berdasarkan apakah suatu kondisi atau ekspresi bernilai benar (true) atau salah (false).

2. Kegunaan fungsi logika IF antara lain:

- Menggunakan kondisi untuk mengambil keputusan dalam program atau spreadsheet.

- Menampilkan hasil yang berbeda berdasarkan kondisi yang diberikan.

- Menghitung nilai berdasarkan kondisi tertentu.

3. Fungsi IF tunggal adalah penggunaan fungsi IF dengan satu kondisi atau ekspresi saja, seperti IF(kondisi, nilai_jika_benar, nilai_jika_salah).

4. Fungsi IF majemuk adalah penggunaan beberapa fungsi IF dalam satu rumus untuk mengatasi beberapa kondisi yang berbeda. Contohnya, IF(kondisi1, nilai1, IF(kondisi2, nilai2, nilai_default)).

5. Fungsi IF bertingkat adalah penggunaan beberapa pernyataan IF yang bersarang (nested) dalam satu rumus. Misalnya, IF(kondisi1, IF(kondisi2, nilai1, nilai2), nilai_default).

6. Lambang-lambang logika pada Microsoft Excel:

- "=" digunakan untuk perbandingan kesetaraan (equal).

- "<" digunakan untuk perbandingan kurang dari (less than).

- ">" digunakan untuk perbandingan lebih dari (greater than).

- "<=" digunakan untuk perbandingan kurang dari atau sama dengan (less than or equal to).

- ">=" digunakan untuk perbandingan lebih dari atau sama dengan (greater than or equal to).

- "<>" digunakan untuk perbandingan tidak sama dengan (not equal to).

7. Beberapa operator logika yang pernah dipelajari adalah "AND" (dan), "OR" (atau), "NOT" (bukan), "IF" (jika).

8. Fungsi logika adalah perangkat matematika atau perangkat lunak yang digunakan untuk mengambil keputusan berdasarkan kondisi atau perbandingan tertentu.

9. Operator logika adalah simbol atau kata kunci yang digunakan untuk menggabungkan kondisi atau ekspresi dalam suatu pernyataan logika. Lambang logika adalah karakter yang digunakan untuk menggambarkan operasi perbandingan seperti kesetaraan, lebih besar, atau kurang dari.

10. Contoh fungsi IF dalam Excel:

Dalam sebuah spreadsheet Excel, Anda dapat menggunakan fungsi IF untuk menentukan apakah suatu siswa lulus ujian dengan rumus seperti IF(nilai >= 60, "Lulus", "Tidak Lulus"). Rumus ini akan menghasilkan "Lulus" jika nilai siswa sama dengan atau lebih besar dari 60, dan "Tidak Lulus" jika nilai di bawah 60.

21. cara mencari dimicrosoft excel rumus IF atau logikanya agar hasilnya menjadi teh manis atau tidak teh manis


Jawaban:

Caranya dg mengkombinasikan fungsi IF dan fungsi OR.

Saya coba memakai Ms. Excel 2016 kak. Rumus di sel F4 yaitu:

=IF(OR(C4="GARAM";D4="GARAM";E4="GARAM");"BUKAN TEH MANIS";"TEH MANIS")

Penjelasan:

Untuk lebih jelasnya, perhatikan gambar berikut ini (ditunjukkan tanda panah merah).

Jawaban:

Untuk membuat TEH MANIS diperlukan bahan-bahan yang cocok yaitu AIR, TEH, dan GULA. Ketiga bahan tersebut harus ada, jika salah satu tidak ada maka tidak bisa disebut dengan TEH MANIS.

Dalam mencari tiap setiap bahan yang dibutuhkan, diperlukan fungsi MATCH. Misal pada nomor 1 :

MATCH("TEH", C4:E4, 0) -> akan memberikan nilai N/A atau Error karena tidak ada nilai "TEH" pada cell C4 sampai E4

MATCH("AIR", C4:E4, 0) -> akan memberikan nilai 1 karena ada nilai "AIR" pada cell C4

MATCH("GULA", C4:E4, 0) -> akan memberikan nilai 1

Lalu menggunakan fungsi AND dan IF untuk mencocokkan bahan-bahan yang dibutuhkan. Gambarannya seperti ini :

jika -> ADA TEH dan ADA AIR dan ADA GULA -> maka "TEH MANIS" -> jika tidak maka "BUKAN TEH MANIS"

dan karena fungsi AND tidak bisa membandingkan nilai error maka nilai error tersebut di ubah menjadi 0 dengan fungsi IFERROR contoh

IFERROR(MATCH("TEH", C4:E4, 0), 0)

Sehingga rumus akhirnya :

=IF(AND(IFERROR(MATCH("TEH", C4:E4, 0), 0), IFERROR(MATCH("AIR", C4:E4, 0), 0), IFERROR(MATCH("GULA", C4:E4, 0), 0)), "TEH MANIS", "BUKAN TEH MANIS")

23. Tuliskan cara menuliskan argumen fungsi logika if pada Microsoft Excel​


Jawaban:

Cara menggunakan fungsi excel IF atau rumus IF excel pada sebuah rumus atau formula excel adalah dengan mengikuti kaidah penulisan rumus berikut:

IF(TesLogika; KondisiTerpenuhi; [KondisiTidakTerpenuhi])

Dari sintaksis di atas, dapat kita ketahui bahwa rumus atau fungsi IF pada excel terdiri dari 3 argumen:

TesLogika : Argumen ini berisi ekspresi logika yang bisa dievaluasi ke dalam nilai TRUE atau FALSE. Argumen ini merupakan kriteria acuan untuk menentukan nilai yang akan dihasilkan oleh rumus excel IF.

KondisiTerpenuhi : Nilai yang akan dihasilkan jika hasil evaluasi TesLogika bernilai TRUE. Argumen ini merupakan nilai yang akan dihasilkan jika kondisi terpenuhi.

KondisiTidakTerpenuhi : (Opsional) Nilai yang ingin kita hasilkan jika argumen TesLogika mengevaluasi ke FALSE. Argumen ini merupakan nilai jika hasil tes logika tidak terpenuhi.

Untuk penjelasan yang lebih lengkap ada di bawah ini.

Penjelasan:

Pengertian Fungsi IF Excel

Bagaimanakah pengertian fungsi IF atau rumus IF excel?

Fungsi IF atau Rumus IF adalah salah satu fungsi Excel dalam kategori atau kelompok logical yang sering digunakan untuk melakukan uji logika tertentu pada rumus microsoft excel.

 

Rumus Excel IF akan menghasilkan sebuah nilai tertentu jika kondisi yang kita tentukan untuk di evaluasi terpenuhi (TRUE) dan akan menghasilkan nilai lainnya apabila uji logika atau kondisi itu tidak terpenuhi (FALSE).

Dengan kata lain fungsi IF excel ini akan melakukan evaluasi terhadap uji logika tertentu kemudian menghasilkan nilai TRUE atau FALSE. Nilai TRUE adalah nilai dimana kondisi tersebut terpenuhi dan nilai FALSE adalah nilai untuk kondisi yang tidak terpenuhi.

TRUE

Benar. Kondisi terpenuhi. Evaluasi uji logika mengasilkan benar.

FALSE

Salah. Kondisi tidak terpenuhi. Evaluasi uji logika salah.

Untuk melakukan uji logika ini, fungsi if pada excel biasanya menggunakan operator perbandingan excel yang sudah kita bahas sebelumnya. yakni menggunakan operator =, >, >= , < atau <= dan lain sebagainya.

Oleh sebab itu sebelum lebih jauh mempelajari fungsi logika ini ada baiknya anda memahami dulu tentang operator Excel yang sudah kita bahas sebelumnya.

Apabila sudah cukup memahami tentang operator perbandingan di excel selanjutnya kita akan belajar tentang bagaimana cara menggunakan rumus if pada excel.

28. tuliskan minimal 4 fungsi logika pada excel beserta contoh penggunaannya​


Jawaban:

Fungsi IF

Fungsi IF ini mungkin adalah salah satu fungsi rumus excel yang paling sering digunakan. Fungsi ini membantu membuat perbandingan logis antara nilai dan perkiraan.

Rumus IF berguna untuk mengembalikan sebuah nilai jika kondisi yang dicek BENAR/SALAH dan mengembalikan sebuah nilai lainnya jika kondisi yang dicek SALAH/FALSE.

Rumus fungsi IF:

=IF (logical_test, value_if_true, value_if_false)

Fungsi OR

Fungsi OR adalah fungsi bawaan di Excel yang dikategorikan sebagai Fungsi Logika. Fungsi OR adalah fungsi logis untuk menguji beberapa kondisi pada saat bersamaan. Rumus OR akan menghasilkan TRUE jika beberapa argument bernilai benar dan FALSE jika semua argument salah.

Rumus fungsi OR:

=OR(logical1; logical2; ...)

Fungsi AND

AND merupakan rumus fungsi logika pada Excel yang berguna untuk menghasilkan nilai TRUE, jika semua argument yang diuji bernilai benar. Jika FALSE, maka semua atau salah satu argument bernilai salah.

Rumus fungsi AND:

=AND (logical1; logical2; ...)

39. cara menggunakan fungsi IF ( logika) dalam microsoft excel


Isi di formula bar seperti ini:
=IF(nama sel,jika benar,jika salah)
Maksudnya:
- nama sel: nama sel yang menjadi acuannya (bisa diisi nama sel seperti A1, atau nama sel dengan tanda operator logika seperti A1>=75)
- jika benar: string (data/angka/tulisan)atau hasilnya jika sel yang kita jadikan acuan benar
- jika salah: string/hasilnya jika sel yang kita jadikan acuan salah
Pada jika benar dan jika salah bisa diisi dengan angka (50%,30%) atau tulisan (tapi kalau tulisan harus diawali dan diakhiri dengan tanda petik ("..."), misal "Lulus","Tidak Lulus")
- tanda , adalah tanda pemisah, bisa diganti dengan ; (tergantung versi excel yang dipakai dan settingan regional di komputermu)

Contoh:
di sel A5 ada nilai 70, sedangkan kkmnya adalah 75, jadi rumusnya memakai:
=IF(A5>=75,"Lulus","Tidak Lulus") maka hasilnya adalah Tidak Lulus
