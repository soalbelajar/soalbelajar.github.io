1. Soal Olimpiade, Mohon Bantuannya Nilai bilangan bulat n sehingga n²– 11n + 63 merupakan bilangan kuadrat adalah ….


Jawab:

n = 38 atau n = -27

Penjelasan dengan langkah-langkah:

n² - 11n + 63 = k²           | x4

4n² - 44n + 252 = 4k²

(2n - 11n)² + 131 = (2k)²

(2k)² - (2n - 11)² = 131

(2k + 2n − 11) . (2k − 2n + 11) = 131

131 adalah bilangan prima

Kasus 1:

2k + 2n − 11 = 131

2k − 2n + 11 = 1

________________ -

4n − 22 = 130

n = (130 + 22)/4

n = 38

Kasus 2:

2k + 2n − 11 = 1

2k − 2n + 11 = 131

________________ -

4n − 22 = -130

n = (-130 + 22)/4

n = -27

Jawaban:

13. bilangan bulat x jika di kalian 11 terletak antara 1500 dan 2000. jika x di kalian 7 terletak antara 970 dan 1275. jika x dikalian 5 terletak antara 960 dan 900. banyaknya bilangan x sedemikian yang habis di bagi 3 sekaligus habis di bagi 5 ada sebanyak ... (soal olimpiade tingkat kabupaten) Makasih :)


1500 < 11x < 2000
970 < 7x < 1275
900 < 5x < 960

maka, bilangan tersebut berada di antara

136 4/11 < x < 181 8/11
138 4/11 < x < 182 1/11
180 < x < 192

sehingga,
HP1 = {137, 138, 139, 140, ...., 181}
HP2 = {139, 140, 141, 142, ...., 182}
HP3 = {181, 182, 183, ...., 192}

HP1 n HP2 n HP3 = {181, 182}

x habis dibagi 3 dan 5, maka x akan habis dibagi 15

181 ≡ 1 mod 15
182 ≡ 2 mod 15

jadi tidak ada bilangan bulat yang memenuhi.

15. Biologi"Katanya Soal Olimpiade"


[tex] Sitologi, Miologi, Kardiologi [/tex]

Pada sistem ECG, terdapat beberapa gambar.

Berdasarkan gambar A, maka
• Titik 1 = Kenaikan titik potensial namun belum menjadi titik potensial maksimum
• Titik 2 = Penurunan titik potensial membran
• Titik 3 = Kenaikan potensial membran
• Titik 4 = Fluktuasi potensial membran
• Titik 5 = Titik kenaikan potensial aksi menjadi potensial aksi membran
• Titik 6 = Titik potensial maksimum (ini peralihan antara depolarisasi dan repolarisasi)
• Titik 7 = Titik ekstrem atau titik minimum (hiperpolarisasi)

Berdasarkan soal,
A. Untuk ventrikel, karena sangat kuat maka menyebabkan titik potensial membran maksimal maka bernomor (6)
B. Untuk atrium, karena tidak terlalu kuat dibanding ventrikel, maka ini tidak dapat menyebabkan titik potensial maksimal, jadi yang nomor (1)
C. Untuk repolarisasi ventrikel (karena sistole sangat kuat) maka lihat yang paling besar persentase penurunannya, maka ini bernomor (7)


Berdasarkan gambar B, maka
• Titik 1 = repolarisasi otot atrium (ciri-ciri yaitu persentase penurunan yang paling menonjol)
• Titik 2 = titik disebelah kiri tidak mengalami penurunan menonjol (ini hanya penutupan katup)
• Titik 3 = titik yang menurun
• Titik 4 = hanya bersifat diastole

Ingat, diastole memiliki range yaitu garis yang panjang dan tidak mengalami kenaikan yang signifikan sedangkan sistole yang diagram kenaikan dan penurunan secara signifikan. Jadi, terjadi sistole sebanyak 5 kali dan diastole sebanyak 5 sehingga terjadi 5 kali siklus karena kiri merupakan setengah dari tekanan diastole dan gambae paling kanan setengah diastole menjadi 1 diastole.

16. soal-soal olimpiade matamatika sma dan caranya​


Jawaban:

diketahui bilangan bulat x dan y dibagi 4, maka bersisa 3.

x=4m+3

y=4n+3

3y=3(4n+3)=4(3n)+9=4(3n+2)+1

x-3y=4m+3-(4(3n+2)+1)

=4m+3-4(3n+2)-1

=4(m-3n-2)+2

jadi jika bilangan x-3y dibagi 4 maka bersisa 2

18. tolong soal olimpiade​


[tex]\large\text{$\begin{aligned}\sf Jawaban:\ \boxed{\:\bf1\:}\end{aligned}$}[/tex]
 

Pembahasan

Diketahui
Untuk bilangan real positif a, b, c, p, q, dan r berlaku:
[tex]\begin{aligned}\frac{a}{p}=\frac{b}{q}=\frac{c}{r}\end{aligned}[/tex]

Ditanyakan
[tex]\begin{aligned}\frac{abc\cdot(p+q)(q+r)(r+p)}{pqr\cdot(a+b)(b+c)(c+a)}={\dots}\end{aligned}[/tex]

PENYELESAIAN

[tex]\begin{aligned}&\frac{a}{p}=\frac{b}{q}=\frac{c}{r}\\&{\implies}\begin{cases}(1)&aq=bp\\(2)&br=cq\\(3)&ar=cp\end{cases}\end{aligned}[/tex]

Kemudian, kita selesaikan.

[tex]\begin{aligned}&\frac{abc\cdot(p+q)(q+r)(r+p)}{pqr\cdot(a+b)(b+c)(c+a)}\\{=\ }&\frac{a(p+q)\cdot b(q+r)\cdot c(r+p)}{p(a+b)\cdot q(b+c)\cdot r(c+a)}\\{=\ }&\frac{(ap+aq)}{(ap+bp)}\cdot\frac{(bq+br)}{(bq+cq)}\cdot\frac{(cr+cp)}{(cr+ar)}\end{aligned}[/tex]
[tex]\begin{aligned}{=\ }&\frac{\overbrace{(ap+bp)}^{\!\!\!\begin{array}{c}(1):bp\to aq\end{array}\!\!\!}}{(ap+bp)}\ \cdot\ \frac{\overbrace{(bq+cq)}^{\!\!\!\begin{array}{c}(2):cq\to br\end{array}\!\!\!}}{(bq+cq)}\ \cdot\ \frac{\overbrace{(cr+ar)}^{\!\!\!\begin{array}{c}(3):ar\to cp\end{array}\!\!\!}}{(cr+ar)}\\{=\ }&1\cdot1\cdot1\\{=\ }&\boxed{\:\bf1\:}\end{aligned}[/tex]

KESIMPULAN

[tex]\begin{aligned}\therefore\ \frac{abc\cdot(p+q)(q+r)(r+p)}{pqr\cdot(a+b)(b+c)(c+a)}=\boxed{\:\bf1\:}\end{aligned}[/tex]
[tex]\blacksquare[/tex]

25. Soal OlimpiadeMisalkan N adalah bilangan bulat terkecil yang bersifat : bersisa 2 jika dibagi 5, bersisa 3 jika dibagi oleh 7, dan bersisa 4 jika dibagi 9. Berapakah hasil penjumlahan digit-digit dari N ?


A = bil bulat positif {1,2,3,...}
N = dibagi 5 sisa 2 ⇒ N = 5k + 2 , k ∈ A  (1)
N = dibagi 7 sisa 3 ⇒ N = 7k + 3 , k ∈ A  (2)
N = dibagi 9 sisa 4 ⇒ N = 9k + 4 , k ∈ A  (3)
misalkan ada suatu bilangan positif m sedemikian sehingga:
9m + 4 = 5k + 2
9m + 2 = 5k, (kelipatan 5)
Untuk m, k ∈ A benar untuk m = 2, 7, 12, 17, ...
misalkan ada suatu bilangan positif l sedemikian sehingga:
9m + 4 = 7l + 3
9m + 1 = 7l, (kelipatan 7)
Untuk m, l ∈ A benar untuk m = 3, 10, 17, ...
Maka N dipenuhi oleh bilangan bulat terkecil m = 17
N = 9m + 4 = 9(17) + 4 = 157
Jumlah digit-digitnya = 1 + 5 + 7 = 13

jadi jumlah digitnya:
= 1 + 5 + 7 = 13
