1. contoh soal bentuk akar beserta jawaban dan penjelasan​


Jawaban:

Tentu, berikut adalah contoh soal bentuk akar beserta jawaban dan penjelasannya:

Soal 1:

Hitunglah akar kuadrat dari 25.

Jawaban 1:

Akar kuadrat dari 25 adalah 5.

Penjelasan 1:

Akar kuadrat dari suatu bilangan adalah bilangan yang jika dipangkatkan dengan 2 akan menghasilkan bilangan tersebut. Dalam kasus ini, kita mencari akar kuadrat dari 25. Jadi, kita mencari bilangan yang ketika dipangkatkan dengan 2 akan menghasilkan 25. Dalam hal ini, 5 adalah akar kuadrat dari 25 karena 5^2 = 25.

Soal 2:

Hitunglah akar pangkat tiga dari 64.

Jawaban 2:

Akar pangkat tiga dari 64 adalah 4.

Penjelasan 2:

Akar pangkat tiga dari suatu bilangan adalah bilangan yang jika dipangkatkan dengan 3 akan menghasilkan bilangan tersebut. Dalam kasus ini, kita mencari akar pangkat tiga dari 64. Jadi, kita mencari bilangan yang jika dipangkatkan dengan 3 akan menghasilkan 64. Dalam hal ini, 4 adalah akar pangkat tiga dari 64 karena 4^3 = 64.

Soal 3:

Hitunglah akar kuadrat dari -16.

Jawaban 3:

Akar kuadrat dari -16 tidak memiliki nilai real.

Penjelasan 3:

Akar kuadrat dari suatu bilangan negatif tidak memiliki nilai real dalam bilangan nyata. Dalam kasus ini, kita mencari akar kuadrat dari -16. Karena -16 adalah bilangan negatif, maka akar kuadratnya tidak memiliki nilai real. Namun, dalam bilangan kompleks, akar kuadrat dari -16 dapat ditulis sebagai 4i atau -4i, di mana i adalah satuan imajiner.

Harap dicatat bahwa contoh soal di atas hanya menggambarkan beberapa situasi umum dalam menghitung akar. Terdapat lebih banyak contoh soal yang dapat dikembangkan, dan jawaban serta penjelasan dapat berbeda tergantung pada konteks yang diberikan.

Penjelasan dengan langkah-langkah:

Berikut adalah beberapa contoh soal bentuk akar beserta jawaban dan penjelasan:

Contoh Soal 1:

Tentukan akar-akar dari persamaan kuadrat berikut: x^2 - 5x + 6 = 0

Jawaban dan Penjelasan:

Dalam hal ini, kita dapat menggunakan metode faktorisasi untuk menyelesaikan persamaan kuadrat ini.

Pertama, kita mencari dua angka (a dan b) yang jika dikalikan akan menghasilkan 6, dan jika ditambahkan akan menghasilkan -5. Dalam kasus ini, angka-angka tersebut adalah -2 dan -3.

Selanjutnya, kita mengganti persamaan menjadi faktor-faktornya: (x - 2)(x - 3) = 0

Ketika kita mengatur setiap faktor menjadi nol, kita mendapatkan akar-akar persamaan: x - 2 = 0 atau x - 3 = 0

Dengan menyelesaikan persamaan ini, kita mendapatkan akar-akar persamaan: x = 2 atau x = 3

Contoh Soal 2:

Tentukan akar-akar dari persamaan kuadrat berikut: 2x^2 + 3x - 2 = 0

Jawaban dan Penjelasan:

Dalam kasus ini, kita tidak dapat langsung menggunakan metode faktorisasi untuk menyelesaikan persamaan kuadrat ini. Oleh karena itu, kita dapat menggunakan rumus kuadrat untuk mencari akar-akarnya.

Rumus kuadrat dapat ditulis sebagai x = (-b ± √(b^2 - 4ac)) / 2a

Dalam hal ini, a = 2, b = 3, dan c = -2

Kemudian, kita masukkan nilai-nilai ini ke dalam rumus kuadrat dan melakukan perhitungan:

x = (-3 ± √(3^2 - 4 * 2 * -2)) / (2 * 2)

x = (-3 ± √(9 + 16)) / 4

x = (-3 ± √25) / 4

x = (-3 ± 5) / 4

Maka, kita memiliki dua akar persamaan: x = (-3 + 5) / 4 atau x = (-3 - 5) / 4

Jadi, akar-akar persamaan ini adalah: x = 1 atau x = -2.5

b. √6 + √54 – √150
c. √32 – √2 + √8
d. √48 – (√27 + √12)


Penyelesaian:
a. Sederhanakan terlebih dahulu √32, yakni:
=> √32 = √(16 × 2)
=> √32 = √16×√2
=> √32 = 4√2
maka:
=> √2 + √32 = √2 + 4√2
=> √2 + √32 = (1 + 4)√2
=> √2 + √32 = 5√2

b. Sederhanakan terlebih dahulu √54 dan √150, yakni:
=> √54 = √(9×6)
=> √54 = √9 × √6
=> √54 = 3√6

=> √150 = √(25×6)
=> √150 = √25 × √6
=> √150 = 5√6

maka:
=> √6 + √54 – √150 = √6 + 3√6 – 5√6
=> √6 + √54 – √150 = (1 + 3 – 5)√6
=> √6 + √54 – √150 = –√6

c. Sederhanakan terlebih dahulu √32 dan √8, yakni:
=> √32 = √(16×2)
=> √32 = √16× √2
=> √32 = 4√2

=> √8 = √(4×2)
=> √8 = √4 × √2
=> √8 = 2√2
maka:
=> √32 – √2 + √8 = 4√2 – √2 + 2√2
=> √32 – √2 + √8 = (4 – 1 + 2)√2
=> √32 – √2 + √8 = 5√2

d. Sederhanakan terlebih dahulu √48, √27 dan √12, yakni:
=> √48 = √(16 × 3)
=> √48 = √16 × √3
=> √48 = 4√3

=> √27 = √(9 × 3)
=> √27 = √9 × √3
=> √27 = 3√3

=> √12 = √(4 × 3)
=> √12 = √4 × √3
=> √12 = 2√3

maka:
=> √48 – (√27 + √12) = 4√3 – (3√3 + 2√3)
=> √48 – (√27 + √12) = 4√3 – (3 + 2)√3
=> √48 – (√27 + √12) = 4√3 – 5√3
=> √48 – (√27 + √12) = (4 – 5)√3
=> √48 – (√27 + √12) = –√3

Jawab:

Pertama, kita tentukan dulu persamaan yang paling sederhana. Dari ketiga persamaan yang ada, persamaan pertama lebih sederhana. Dari persamaan pertama, nyatakan variabel x sebagai fungsi y dan z sebagai berikut.
