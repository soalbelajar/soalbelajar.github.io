4. salah satu cara untuk memeriksa data yang telah di update dengan menggunakan


Meng-Update(Mengubah /Memperbarui)DData

Ada  beberapa cara untuk melakukan update data pada suatu aplikasi basis  data. Pada Microsoft Access, update data dapat dilakukan melalui tabel,query, dan form.

1.Update data melalui tabel

Untuk meng-update data melalui tabel ,langkah pertama , yaitu bukalah tabel data yang datanya ingin Anda perbarui .Misalnya  anda akan mengupdate data pada tabel ‘’Daftar Siswa”.

Langkah-langkah sebagai berikut:

a)Buka basis data yang telah Anda buat sebelumnya . Start> All program >Microsoft Office>Microsoft Access 2007.

b)Lalu klik Microsoft Office Button>Open>Name File Anda Tugas1. Accdb>klik open,

c)Buka kembali data Daftar Siswa ,kemudian buka tabel Daftar .caranya dobel klik pada data Daftar Siswa yang telah Anda simpan sebelumnya .Maka akan terlihat tampilan  seperti berikut dan Anda sudah bisa  melalui update datanya .

d) Andaikata data yang telah Anda buat tersebut di update dengan menambahkan dua record baru

e)Masukkan data baru secara berturut turut pada setiap record

f)Setelah data di-update, klik menu Office Button dan pilih Save. Kemudian tutup tabel dengan mengklik menu Office Button dan Close.

12. sebutkan langkah langkah mengupdate data melalui tabel​


Memilih field dan RecordLangkah pertama yang harus dilakukan adalah memiliki data sebelum melakukan editing pada sebuah jendela data sheet. Untuk memilih data atau record pada sebuah jendela datasheet

Memindahkan Insertion PointMemindahkan insertion point atau data record dalam sebuah datasheet dapat dilakukan dengan menggunakan fasilitas yang ada pada perintah Go To dari menu Edit, panah atas, panah bawah, Page Up atau Page Down. Selain itu kita juga dapat memindahkan posisi record aktif dengan memilih tombol navigasi yang ada pada bagian bawah dari jendela datasheet sebuah table

Menambahkan record Penambahan record dalam sebuah database dapat dilakukan jika telah dibuat sebuah design database. Penambahan record ini dapat dilakukan dalam query maupun dalam form sebuah database.
