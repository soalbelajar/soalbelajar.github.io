Conditional sentence: kalimat pengandaian.

Type 0: kalimat yang menjelaskan fakta atau pernyataan umum.

Rumus: If + simple present tense , simple present tense

Contoh:

If we heat the water, it boils. Jika kita memanaskan air, dia akan mendidih.If it rains, it is wet. Jika hujan, maka akan basah.If I eat a lot, I gain weight. Jika saya makan banyak, saya akan menambah berat badan. Type 1: situasi yang mungkin terjadi di masa mendatang atau masa sekarang.

Rumus: If + simple present tense, subject + will + bare infinitive

Contoh:

If it rains, I will not come to your house. Kalau hujan, saya tidak akan datang ke rumah kamu. If I study hard, I will pass the exam. Jika saya belajar giat, saya akan lolos ujian. Type 2: Kejadian yang tidak mungkin terjadi di masa sekarang atau mendatang.

Rumus: If + simple past tense, subject + would + bare infinitive

Contoh:

If I won the competition, I would buy a new cellphone. Jika saya memenangkan lomba, saya akan membeli HP baru. If I had a lot of money, I would buy a new car. Jika saya mempunyai banyak uang, saya akan membeli mobil baru. Type 3: Sesuatu yang tidak terjadi di masa lalu.

Rumus: If + past perfect tense, subject + would have + verb 3

Contoh:

If I had studied hard, I would have passed the exam. Jika saya telah belajar keras, saya akan lolos ujian.

PEMBAHASAN:

Semua soal di atas menggunakan pola kalimat present perfect tense. Perhatikan penjelasan tentang present perfect tense berikut ini!

Present perfect tense merupakan suatu tense yang menggambarkan suatu kejadian yang terjadi sebelum satu spesifik waktu di masa lalu tetapi masih berlanjut sampai sekarang. Tense ini terdiri dari gabungan aspek perfect (sebelum satu spesifik waktu di masa lampau) dan bingkai waktu present (masa sekarang).  

Selain itu, tense ini juga bisa menggambarkan suatu kejadian yang dimulai dan berakhir di masa lampau tetapi efeknya masih dirasakan sampai saat sekarang. Lebih singkatnya, tense ini digunakan sebagai penghubung antara masa lalu dan masa sekarang.  

Rumus Present Perfect Tense  

Present perfect tense dibentuk dengan auxiliary verb have atau has dan past participle (verb 3). Have berpasangan dengan subjek I, You, We, dan They sedangkan Has digunakan untuk subjek He, She, dan It.  

Positive

I/You/We/They + have + verb 3  He/She/It + has + verb 3  

Negative

I/You/We/They + have + not + verb 3  He/She/It + has + not + verb 3  

Interrogative

Have + I/You/We/They + verb 3?  Has + He/She/It + verb 3?
