13. Aplikasi yang dipakal Tindakan Percobaan Aplikasi pengolah lembar kerja Bukalah sebuah file .docx yang telah disiapkan guru. Simpan sebuah file .docx menjadi .rtf dan buka file .rtf tersebut. Aplikasi pengolah Simpan sebuah file.docx menjadi kata pdf dan buka file .pdf tersebut. Simpan sebuah file .docx menjadi .xlsx, dengan menulis langsung nama ekstensinya saat Save As. Buka file .xlsx tersebut. Buka sebuah file lembar kerja (.xlsx). Simpan sebuah file .xlsx menjadi .txt dan buka kembali file .txt tersebut dengan aplikasi pengolah lembar kerja. Simpan sebuah file .xlsx menjadi .csv dan buka kembali file .csv tersebut dengan aplikasi pengolah lembar kerja. Simpan sebuah file .xlsx menjadi .pdf dan buka kembali file .pdf tersebut dengan aplikasi pengolah lembar kerja. Simpan sebuah file .xlsx menjadi .docx Apa yang terjadi? Apa kesimpulan kalian? Mengapa demikian?​


Percobaan dengan word processor atau pengolah kata:

Jika kita menyimpan sebuah file .docx menjadi .rtf dan membuka file .rtf tersebut, yang terjadi adalah file .docx akan dikonversi ke format .rtf, yang merupakan rich text format.  Jika kita menyimpan sebuah file .docx menjadi .pdf dan membuka file .pdf tersebut, yang terjadi adalah file .docx akan dikonversi ke format .pdf, yang merupakan format dokumen portabel (portable document format). Jika kita menyimpan sebuah file .docx menjadi .xlsx, dengan menulis langsung nama ekstensinya saat Save As. Buka file .xlsx tersebut, yang terjadi adalah file .docx akan disimpan dengan ekstensi .xlsx, tetapi tidak akan dikonversi ke format .xlsx, yang merupakan format spreadsheet (lembar kerja).

Pembahasan

Ada pun bila dilakukan percobaan dengan aplikasi spreadsheet atau pengolah lembar kerja:

Jika kita membuka sebuah file lembar kerja (.xlsx) dengan aplikasi pengolah lembar kerja, yang terjadi adalah file .xlsx akan terbuka dengan normal, karena formatnya sesuai. Jika kita menyimpan sebuah file .xlsx menjadi .txt dan membuka kembali file .txt tersebut dengan aplikasi pengolah lembar kerja, yang terjadi adalah file .xlsx akan dikonversi ke format .txt, yang merupakan format teks biasa (plain text). Jika kita simpan sebuah file .xlsx menjadi .csv dan membuka kembali file .csv tersebut dengan aplikasi pengolah lembar kerja, yang terjadi adalah file .xlsx akan dikonversi ke format .csv, yang merupakan format comma separated values. Jika kita simpan sebuah file .xlsx menjadi .pdf dan membuka kembali file .pdf tersebut dengan aplikasi pengolah lembar kerja, yang terjadi adalah file .xlsx akan dikonversi ke format .pdf.Jika kita simpan sebuah file .xlsx menjadi .docx, yang terjadi adalah file .xlsx akan disimpan dengan ekstensi .docx, tetapi tidak akan dikonversi ke format .docx.Pelajari lebih lanjutMateri tentang beberapa aspek ilmu komputer https://brainly.co.id/tugas/16652646Materi tentang jenis software https://brainly.co.id/tugas/2300113Materi tentang beberapa software komputer https://brainly.co.id/tugas/6674111Detail Jawaban

16. cara agar bisa merubah file dokumen menjadi file ebook dengan format pdf ?


Cara agar bisa merubah file dokumen menjadi file ebook dengan format pdf adalah :

Buka file berekstensi dokumen yang akan Anda ubah menjadi file berekstensi PDF (dalam kasus ini penulis menggunakan aplikasi Ms. Word 2013)Buka tab file, dan pilih eksporPilih "Buat PDF/XPS"Lalu beri nama file berekstensi PDF yang Anda inginkanDan klik "terbitkan"Selamat file berekstensi dokumen Anda sudah menjadi file berformat PDF

PEMBAHASAN :

Sebelum menjawab soal tersebut, mari kita mengenal terlebih dahulu file yang berekstensi pdf ^_^

→ Portable Document Format atau biasa disingkat dengan PDF, merupakan suatu ekstensi yang dikembangkan oleh perusahaan Adobe Inc pada tahun 1993.

→ Ektensi PDF ini memungkinkan user untuk dapat merubah baik file berupa dokumen (doc atau docx) dan gambar menjadi satu file berekstensi PDF yang praktis dengan kompabilitas tinggi.

→ Untuk dapat membuka file PDF, diperlukan aplikasi reader khusus seperti Adobe reader, Foxit PDF, Nitro PDF, dan banyak aplikasi pihak ke 3 lainnya.

→ Saat ini tingkat penggunaan file berekstensi PDF sangat tinggi, disebabkan oleh beberapa kelebihan yang dimilikinya, seperti :

membutuhkan ruang penyimpanan (memori) yang relatif kecil dibanding ekstensi lainnyafile dokumen berupa tulisan atau grafik akan dianggap sebagai gambar, sehingga memudahkan untuk perpindahan halaman saat membaca ebooktidak adanya kekhawatiran terhadap kemungkinan salah tekan atau perubahan konten ebook yang tidak disengaja, karena file pdf akan menganggap isinya sebagai sebuah gambar yang solid, dan hanya bisa di edit saat memiliki aplikasi pdf editor yang rata-rata berbayarlebih tahan akan kemungkinan kerusakan dokumen, di banding format docx

Nah, setelah tau kelebihan file berekstensi PDF, lalu timbul pertanyaan bagaimana merubah file berformat dokumen (doc), menjadi PDF ?

Langkah yang dapat dilakukan adalah sebagai berikut :

Buka file berekstensi dokumen yang akan Anda ubah menjadi file berekstensi PDF (dalam kasus ini penulis menggunakan aplikasi Ms. Word 2013)Buka tab file, dan pilih eksporPilih "Buat PDF/XPS"Lalu beri nama file berekstensi PDF yang Anda inginkanDan klik "terbitkan"Selamat file berekstensi dokumen Anda sudah menjadi file berformat PDF :)

*Jika Anda memiliki aplikasi pihak ke 3, seperti foxit phantom PDF, maka pengubahan file dokumen atau gambar akan menjadi lebih mudah, yakni hanya dengan klik kanan kursor pada file yang akan diubah ekstensinya, lalu pilih "convert to pdf"

37. apa itu surat lamaran kerja?buatlah contoh surat lamaran kerja ​


Surat lamaran kerja adalah sebuah dokumen yang dibuat oleh seseorang yang sedang mencari pekerjaan untuk mengajukan diri sebagai calon karyawan di suatu perusahaan atau organisasi. Surat ini berisi informasi tentang latar belakang, pengalaman, kualifikasi, dan motivasi calon karyawan. Berikut adalah contoh sederhana surat lamaran kerja:

```

[Alamat Pengirim]

[Email]

[No. Telepon]

[Tanggal]

[Alamat Tujuan]

[Perusahaan]

[Alamat Perusahaan]

Perihal: Lamaran Pekerjaan sebagai [Posisi yang Dilamar]

Yth. [Nama Penerima],

Saya yang bertanda tangan di bawah ini,

Nama: [Nama Lengkap]

Tempat/Tanggal Lahir: [Tempat, Tanggal Lahir]

Alamat: [Alamat Tinggal]

Pendidikan Terakhir: [Gelar Pendidikan Terakhir]

No. Telepon: [Nomor Telepon]

Email: [Alamat Email]

Dengan ini ingin mengajukan diri sebagai calon [Posisi yang Dilamar] di [Perusahaan]. Saya sangat tertarik untuk bergabung dengan perusahaan yang bergerak di bidang [Bidang Perusahaan] ini dan berkontribusi dalam mencapai tujuan perusahaan.

Selama [Jangka Waktu] tahun bekerja di [Nama Perusahaan Sebelumnya], saya telah mengembangkan keterampilan dalam [Poin-poin Kualifikasi yang Relevan]. Saya juga memiliki kemampuan dalam [Keterampilan Lain yang Relevan] yang dapat saya bawa untuk mendukung tim di [Perusahaan].

Saya sangat antusias terhadap kesempatan untuk belajar lebih lanjut di [Perusahaan] dan berkontribusi pada tim yang sudah luar biasa ini. Saya telah melampirkan daftar riwayat hidup saya sebagai informasi lebih lanjut tentang latar belakang pendidikan dan pengalaman kerja saya.

Terima kasih atas perhatian Anda terhadap lamaran saya. Saya sangat berharap dapat memiliki kesempatan untuk berbicara lebih lanjut dalam wawancara. Saya siap menghadiri wawancara kapan saja yang Anda tentukan.

Hormat saya,

[Tanda Tangan Ditulis]

[Nama Lengkap]

Lampiran: Daftar Riwayat Hidup

```

Pastikan untuk menyesuaikan konten surat lamaran dengan informasi pribadi dan detail posisi yang dilamar.
