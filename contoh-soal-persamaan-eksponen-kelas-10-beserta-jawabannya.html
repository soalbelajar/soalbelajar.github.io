6. persamaan eksponen kelas 10​


Penjelasan dengan langkah-langkah:

h(x)^f(x) = h(x)^g(x)

Syarat - syaratnya :

• f(x) = h(x)

• h(x) = 1

• h(x) = 0 (untuk f(x), g(x) > 0)

• h(x) = -1 (untuk nilai f(x) dan g(x) keduanya genap atau keduanya ganjil)

(x - 3)ˣ⁻⁶ = (x - 3)³

h(x) = x - 3, f(x) = x - 6, g(x) = 3

• untuk f(x) = g(x)

x - 6 = 3

x = 3 + 6

x = 9

• untuk h(x) = 1

x - 3 = 1

x = 1 + 3

x = 4

• untuk h(x) = 0

x - 3 = 0

x = 3

coba kita cek :

f(3) = 3 - 6 = -3 (tidak memenuhi)

karena nilai f(x) < 0, berarti x = 3 tidak memenuhi

• untuk h(x) = -1

x - 3 = -1

x = -1 + 3

x = 2

kita cek :

f(2) = 2 - 6 = -4

g(2) = 3

karena nilai f(x) genap dan g(x) ganjil, berarti nilai x = 2 tidak memenuhi

15. contoh soal eksponen kelas 10bikin sama jawabannya​


Jawaban:

Diketahui akar-akar persamaan dari persamaan eksponensial 32y + 1 – 28 . 3y + 9 = 0 adalah y1 dan y2. Apabila nilai akar y1 > y2, tentukan berapakah nilai 4y1 – y2 …

Pembahasan:

32y + 1 – 28 . 3y + 9 = 0

3 . (3y) – 28 . 3y + 9 = 0

Agar solusi dari persamaan eksponensial bisa didapatkan maka 3^y harus diubah menjadi variabel sendiri misalkan X. Persamaan eksponensial menjadi:

3X² - 28X+9 = 0

(X – 9) (3X – 1) = 0

Sehingga nilai variabel dari persamaan eksponensial adalah X = 9 dan X = 1/3. Akar kedua persamaan adalah:

a) X = 3^y

3² = 3^y

y = 2

b) X = 3^y

1/3 = 3^y

3ˉ¹ = 3^y

y = -1

Maka akar y1 = 2 dan y2 = -1 sehingga nilai 4y1 – y2 bisa dihitung dengan mensubstitusikan akar-akarnya:

4y1 – y2

= 4 (2) - (-1)

= 8 + 1

= 9

20. Bantu jawab kk sekalian tentang persamaan Eksponen, Mtk peminatan kelas 10.


Pembahasan Soal

Soal Nomor 1]

[tex]\frac{1}{27^{3x - 7}} =\sqrt{3^{2 - 2x}}[/tex]

Penyelesaian :

[tex]\frac{1}{27^{3x - 7}} =\sqrt{3^{2 - 2x}}[/tex]

[tex]\frac{1}{27^{3x - 7}} = (3^{2 - 2x})^{\frac{1}{2}}[/tex]

[tex]({27^{3x - 7}})^{-1} = (3)^{\frac{2 - 2x}{2}}[/tex]

[tex]27^{-3x + 7}} = (3)^{1 - x}[/tex]

[tex](3^3)^{-3x + 7}} = (3)^{1 - x}[/tex]

[tex]3^{(-9x + 21)} = 3^{(1 - x)}[/tex]

Maka :

-9x + 21 = 1 - x

 -9x + x = 1 - 21

       -8x = -20

           x = [tex]\frac{20}{8}[/tex]

              = [tex]\frac{5}{2}[/tex] [tex]\boxed{\bold{Jawaban \ E}}[/tex]


Soal Nomor 2]

[tex]\frac{(9+\sqrt{5})(2\sqrt{5}+1)}{(\sqrt{5}+1)}[/tex]

Penyelesaian :

= [tex]\frac{(9+\sqrt{5})(2\sqrt{5}+1)}{(\sqrt{5}+1)} \times \frac{(\sqrt{5}-1)}{(\sqrt{5}-1)}[/tex]

= [tex]\frac{(18\sqrt{5}+9+10+\sqrt{5})}{(\sqrt{5}+1)} \times \frac{(\sqrt{5}-1)}{(\sqrt{5}-1)}[/tex]

= [tex]\frac{(18\sqrt{5}+19+\sqrt{5})}{(\sqrt{5}+1)} \times \frac{(\sqrt{5}-1)}{(\sqrt{5}-1)}[/tex] Lakukan Perkalian

= [tex]\frac{90-18\sqrt{5}+19\sqrt{5}-19+5-\sqrt{5}}{(5-1)}[/tex]

= [tex]\frac{76}{4}[/tex]

= 19 [tex]\boxed{\bold{Jawaban \ B}}[/tex]


Soal Nomor 3]

[tex]\frac{2^{(n+2)}\times 6^{(n - 4)}}{12^{(n-1)}}[/tex]

Penyelesaian :

= [tex]\frac{2^{(n+2)}\times 6^{(n - 4)}}{12^{(n-1)}}[/tex]

= [tex]\frac{2^{n}2^{2}\times 6^{n}6^{-4}}{12^{n}12^{-1}}[/tex] Bilangan dengan pangkat negatif ubah dalam bentuk pangkat positif

= [tex]\frac{2^{n}2^{2}\times 6^{n}12^{1}}{12^{n}6^{4}}[/tex]

= [tex]\frac{2^{n}\times4\times 6^{n}\times12}{12^{n}\times1296}[/tex]

= [tex]\frac{(2\times6)^{n}\times(4\times12)}{12^{n}\times1296}[/tex]

= [tex]\frac{12^{n}\times48}{12^{n}\times1296}[/tex]

= [tex]\frac{48}{1296}[/tex]

= [tex]\frac{1}{27}[/tex] [tex]\boxed{\bold{Jawaban \ A}}[/tex]

21. Bantu jawab kk sekalian tentang persamaan Eksponen, Mtk peminatan kelas 10.


Pembahasan

Soal Nomor 4]

[tex](\frac{1}{4})^{x - 1} = \sqrt[3]{2^{3x+1}}[/tex]

Penyelesaian :

[tex](\frac{1}{4})^{x - 1} = \sqrt[3]{2^{3x+1}}[/tex]

[tex](\frac{1}{4})^{x - 1} ={2^{(\frac{3x+1}{3})}}[/tex]

[tex](\frac{1}{2^2})^{x - 1} ={2^{(\frac{3x+1}{3})}}[/tex]

[tex](2^{-2})^{(x - 1)} ={2^{(\frac{3x+1}{3})}}[/tex]

[tex](2^{-2x + 2})={2^{(\frac{3x+1}{3})}}[/tex]

Maka :

[tex]-2x + 2 = \frac{3x+1}{3}[/tex]

(-2x + 2) x 3 = 3x + 1

-6x + 6 = 3x + 1

-6x - 3x = 1 - 6

       -9x = -5

           x = [tex]\frac{5}{9} \ \ \boxed{\bold{Jawaban \ C}}[/tex]


Soal Nomor 5]

[tex]2^{2x + 2} = \frac{1}{\sqrt{8^{x+1}}}[/tex]

Penyelesaian :

[tex]2^{2x + 2} = \frac{1}{\sqrt{(2^3)^{(x+1)}}}[/tex]

[tex]2^{2x + 2} = \frac{1}{\sqrt{2^{(3x+3)}}}[/tex]

[tex]2^{2x + 2} = \frac{1}{2^{\frac{(3x+3)}{2}}}[/tex]

[tex]2^{2x + 2} = 2^{(-\frac{3x+3}{2})}[/tex]

[tex]2^{2x + 2} = 2^{(\frac{-3x-3}{2})}[/tex]

Maka :

2x + 2 = [tex]\frac{-3x-3}{2}[/tex]

(2x + 2) x 2 = -3x - 3

        4x + 4 = -3x - 3

       4x + 3x = -3 - 4

               7x = -7

                 x = -7 : 7

                 x = -1 [tex]\boxed{\bold{Jawaban \ B}}[/tex]


Soal Nomor 6]

Perhatikan persamaan kedua untuk mencari nilai x - y

[tex]2^{(x-y)}=16[/tex]

[tex]2^{(x-y)}=2^4[/tex]

Maka :

x - y = 4 [tex]\boxed{\bold{Jawaban \ D}}[/tex]


Soal Nomor 7]

[tex](\sqrt{2})^{6x - 4} = (\frac{1}{4})^{x - 9}[/tex]

Penyelesaian :

[tex](2^{\frac{1}{2}})^{6x - 4} = (\frac{1}{2^2})^{x - 9}[/tex]

[tex](2)^{3x - 2} = (2^{-2})^{x - 9}[/tex]

[tex](2)^{3x - 2} = 2^{-2x + 18}[/tex]

Maka :

3x - 2 = -2x + 18

3x + 2x = 18 + 2

       5x = 20

         x = 20 : 5

         x = 4 [tex]\boxed{\bold{Jawaban \ E}}[/tex]

32. contoh soal dan jawaban tentang persamaan eksponen


Penjelasan dengan langkah-langkah:

**Soal 1**

Tentukan nilai x yang memenuhi persamaan eksponen 2x - 1 = 8.

**Penyelesaian**

Kita samakan basis kedua ruas persamaan, yaitu 2.

```

2x - 1 = 8

```

```

2x = 9

```

```

x = 4.5

```

Jadi, nilai x yang memenuhi persamaan tersebut adalah **4.5**.

**Soal 2**

Tentukan nilai x yang memenuhi persamaan eksponen 3x^2 - 2x - 3 = 0.

**Penyelesaian**

Kita faktorkan persamaan tersebut sebagai berikut:

```

(3x + 1)(x - 3) = 0

```

Jadi, x = -1/3 atau x = 3.

**Soal 3**

Tentukan nilai x yang memenuhi persamaan eksponen 2^x = 1/128.

**Penyelesaian**

Kita samakan basis kedua ruas persamaan, yaitu 2.

```

2^x = 1/128

```

```

x = -7

```

Jadi, nilai x yang memenuhi persamaan tersebut adalah **-7**.

**Soal 4**

Tentukan nilai x yang memenuhi persamaan eksponen (2x + 1)^3 = 8.

**Penyelesaian**

Kita ubah persamaan tersebut ke bentuk perkalian, yaitu:

```

(2x + 1)^3 = 2^3 * 1

```

```

(2x + 1)^3 = 8

```

```

(2x + 1) = 2

```

```

x = -1/2

```

Jadi, nilai x yang memenuhi persamaan tersebut adalah **-1/2**.

**Soal 5**

Tentukan nilai x yang memenuhi persamaan eksponen (x - 1)^2 = 9.

**Penyelesaian**

Kita samakan basis kedua ruas persamaan, yaitu x - 1.

```

(x - 1)^2 = 9

```

```

x - 1 = 3

```

```

x = 4

```

Jadi, nilai x yang memenuhi persamaan tersebut adalah **4**.

2(4^x) - 5(2^x) + 2 = 0

(2^x)^2 - 5(2^x) + 2 = 0

(2^x - 1)(2^x - 2) = 0

2^x - 1 = 0 atau 2^x - 2 = 0

2^x = 1 atau 2^x = 2

x = 0 atau x = 1
