18. 1000 mL NH4OH 0,02 M di campur dengan 777mL HNO3O 0,02M. Kb=10pangkat -5


mmol NH4OH = 1000 x 0,02 = 20 mmol
mmol HNO3 = 777 x 0,02 = 15,54 mmol

   NH4OH + HNO3 => NH4NO3 + H2O
a : 20            15,54
b : 15,54        15,54       15,54         15,54
s :  4,46             -           15,54        15,54

[OH-] = Kb x mol basa / mol garam
[OH-] = 10⁻⁵ x 4,46 / 15,54
[OH-] = 2,87 x10⁻⁶
pOH = - log [OH-]
pOH = - log 2,87 x10⁻⁶
pOH = 6 - log 2,87

pH = 14 - pOH
pH = 14 - (6 - log 2,87)
pH = 8 + log 2,87
pH = 8 + 0,46 = 8,46

35. 1000 KB sama dengan berapa pixel data digital


Jawaban:

Tidak ada hubungan langsung antara KB dan pixel data digital tanpa informasi tambahan.

Penjelasan:

Tidak ada konversi langsung antara ukuran berkas digital (dalam KB, kilobyte) dan ukuran pixel data digital. KB (kilobyte) adalah unit ukuran untuk kapasitas penyimpanan data, sedangkan pixel adalah unit ukuran untuk resolusi gambar atau layar.

Ukuran pixel dari sebuah gambar atau layar akan bergantung pada resolusi gambar tersebut. Resolusi digunakan untuk mengukur jumlah piksel horizontal dan vertikal dalam gambar. Misalnya, gambar dengan resolusi 1920x1080 berarti memiliki 1920 piksel lebar dan 1080 piksel tinggi.

Jika Anda ingin mengonversi ukuran berkas gambar digital (dalam KB) ke resolusi pikselnya, Anda perlu tahu lebih banyak informasi tentang gambar tersebut, seperti format gambar, tingkat kompresi, dan faktor-faktor lainnya yang dapat memengaruhi ukuran berkas. Tidak ada hubungan langsung antara KB dan pixel data digital tanpa informasi tambahan.
