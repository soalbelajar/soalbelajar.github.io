3. apa yang membedakan antara conditional type 2 dan conditional sentence type 3 ???


Conditional type 2 menggunakan rumus

If + S + V2, S + would + V1

Contohnya:

If I came, she would be mad.

Conditional type 3 menggunakan rumus

If + S + had + V3, S + would have + V3

Contohnya:

If hadn't been sick, I wouldn't have missed the lesson.
conditional type 3 utk kejadian masa lampau atau simple past tense, kejadian yang harus sudah dipenuhi di masa lalu. Rumus Conditional Sentence Type 3
If +  Subjek + Had + V3 +S+ M.2 + Have + V.3

conditional type 2 ini digunakan utk kejadian di masa sekarang atau present yang akan terjadi jika kondisi yang ada berbeda.
Rumus Conditional Sentence Type 2
If + Subjek +V.2/Were +S + M2 + V.1M2 (Modal bentuk 2) : Would,could

16. Conditional sentence Type 3


A conditional sentence type 3 is used to talk about hypothetical situations and their possible outcomes in the past. It is formed by using the past perfect tense in the "if clause" and the perfect conditional tense (would have + past participle) in the main clause.

The structure of a conditional sentence type 3 is as follows:

If + past perfect tense, would have + past participle.

Here are a few examples:

1. If I had studied harder, I would have passed the exam.

(Implies that the person did not study hard, and as a result, they did not pass the exam.)

2. If she had arrived on time, we would have caught the train.

(Implies that she did not arrive on time, and as a result, they missed the train.)

3. If it hadn't rained, we would have gone for a walk.

(Implies that it did rain, and as a result, they didn't go for a walk.)

In conditional sentence type 3, the "if clause" refers to a past event that did not happen or a condition that was not fulfilled, which would have resulted in a different outcome in the main clause.


17. ubahlah menjadi conditional sentence type 3​


CONDITIONAL SENTENCE

1. If you had studied for the test, you would have passed it.

2. If we had gone to the cinema, we would have seen my friend Jacob.

3. If they had listened to me, we would have been home earlier.

4. If it had not started to rain, we would have walked to the beach.

5. We would have swum in the sea if there hadn't been so many sharks there.

6. If she had taken the bus, she wouldn't have arrived on time.

EXPLANATION

Conditional tipe 1 [First conditional] menjabarkan keinginan yang kemungkinan terjadi saat ini ataupun nanti-nanti (real possibility) berdasarkan prediksi. [True in the present/future]

Rumus

If+S+verb1(s/es)..., S+will+verb1......

Conditional tipe 2 [Second conditional] menjabarkan keinginan yang kemungkinan kecil terwujud atau diragukan terjadi saat ini atau mendatang. [Untrue in the present/future]

Nominal sentence pada conditional tipe 2 menggunakan tobe "were" untuk subyek tunggal dan jamak.

Rumus :

If+S+verb2 ......, S+would+verb1......

Conditional tipe 3 [Third conditional] menjabarkan keinginan yang tidak terwujud di waktu lampau. Pengulasannya saat ini hanyalah sebagai dampak dari kejadian di waktu lampau. [Untrue in the past]

Rumus :

If+S+had+verb3...., S+would+have+verb3....

PENJELASAN

=> CONDITIONAL SENTENCE TYPE 1

First conditional sentence merupakan bentuk kalimat pengandaian yang digunakan ketika hasil atau konsekuensi memiliki kemungkinan terjadi di masa depan. Maka dari itu, bentuk kalimat dari conditional sentence tipe pertama ini memiliki bentuk kalimat simple future alias kalimat yang akan datang. Rumus dari kalimat ini adalah: if + simple present, simple future “will”.

=> CONDITIONAL SENTENCE TYPE 2

Second conditional sentences merupakan tipe kalimat pengandaian yang digunakan ketika hasil atau konsekuensi hanya memiliki harapan terwujud walaupun kemungkinannya sangat kecil. Dalam bentuk kalimat ini, rumus yang digunakan adalah if + simple past/were, would/could/might + bare infinitive. Penggunaan would/could/might berfungsi untuk menjelaskan seberapa besar sebuah kondisi itu akan terjadi.

=> CONDITIONAL SENTENCES TYPE 3

Third conditional sentence merupakan sebuah kalimat pengandaian ketika sebuah kondisi tidak mungkin terwujud sama sekali. Dalam tipe kalimat pengandaian yang ketiga ini, bentuk kalimat menggunaka past perfect yang dilengkapi dengan modal auxiliary seperti would, could, dan should. Rumus yang digunakan adalah if + past perfect, would/should/could/might + have + past participle.

33. kak bantu Ubah menjadi 3 Type Conditional sentence soalnya mau dikumpulin​


CONDITIONAL SENTENCES

1)

(I) I will go to school if there is no corona virus.

(II) I would go to school if there were no corona virus

(III) I would have gone to school if there had been no corona virus

2)

(I) We will buy a luxurious house if we are rich

(II) We would buy a luxurious house if we were rich

(III) We would have bought a luxurious house if we had been rich.

3)

(I) If I don't eat spicy meat ball, my stomach won't hurt.

(II) If I didn't eat spicy meat ball, my stomach would not be hurt

(III) If I hadn't eaten spicy meat ball, my stomach would not have been hurt

EXPLANATION

Conditional sentence adalah kalimat pengandaian, di mana suatu keinginan dapat terwujud apabila persyaratannya terpenuhi. Conditional sentence terdiri dari if clause dan main clause/result clause. If clause menggambarkan kondisinya, dan main clause menggambarkan konsekuensinya. Apabila kondisinya/situasinya adalah pasti benar, maka konsekuensinya akan terwujud. Conditional sentence terbagi dalam tiga tipe, yaitu 1, 2 dan 3.

Conditional tipe 1 menjabarkan keinginan yang kemungkinan terjadi saat ini ataupun nanti-nanti (real possibility) berdasarkan prediksi.

Rumus

If+Simple present, Simple future

If+S+verb1(s/es)..., S+will+verb1......

Conditional tipe 2 menjabarkan keinginan yang kemungkinan kecil terwujud atau diragukan terjadi saat ini atau mendatang.

Nominal sentence pada conditional tipe 2 menggunakan tobe "were" untuk subject tunggal dan jamak.

Rumus :

If+Simple past, Past future

If+S+verb(s/es)...., S+would+verb1......

Conditional tipe 3 menjabarkan keinginan yang tidak terwujud di waktu lampau. Pengulasannya saat ini hanyalah sebagai dampak dari kejadian di waktu lampau.

Rumus :

If+Past perfect, Past future perfect

If+S+had+verb3...., S+would+have+verb3....
