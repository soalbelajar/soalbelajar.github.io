3. matriks soal latihan sbmptn


det(AC) = det(A)*det(C) = 0

Karena C memiliki invers, maka det(C) ≠ 0, maka det(A) haruslah 0

sehingga,

x^2 - 2 = 0
x = ±√2

juga det(B) = 0

sehingga,

(-1) - (-y)(3) = 0
-1 + 3y = 0
y = 1/3

sehingga,

2x^2 + 27y^3 = 2(√2)^2 + 27(1/3)^3 = 4 + 1 = 5

16. LATIHAN SOAL SBMPTN! Matematika Dasar Sebuah matriks A dengan ordo 2016x2016 memiliki unsur aij yang setiap unsurnya merupakan hasil penjumlahan dari i+j Berapakah jumlah unsur diagonalnya? A. (2016)(4032) B. (2016)(4031) C. (2017)(4032) D. (2017)(4031) E. (2016)(2017)(4032) Jawablah dengan disertai alasan. Terima kasih dan selamat menjawab.


Jumlahnya diagonal utama membentuk deret barisan aritmatika yaitu
====>  2 +4+6+8+10+...+4032

Jadi
S2016
= 2016/2 (a+Un)
= 2016/2 (2 + 4032)
= 2016 . 4034 / 2
= (2016)(2017)

Untuk diagonal samping
2017 + 2017 + 2017 + .... + 2017
Jumlahnya
= 2016 x 2017

Jadi jumlahnya
= 2 x 2016 x 2017
= (2017)(4032)

Jadi jawabannya C. 2017.4032
