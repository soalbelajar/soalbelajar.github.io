9. Menentukan Persamaan Kuadrat yang Melatih Ketiga Titik. Soal: Suatu grafik yang melalui titik (5, 0) (1, 4) (0, 3) . Tentukan persamaannya! minta tolong ya kak, menggunakan rumus Y=aX² + bX + c terima kasih. ​


Jawaban:

m = (y₂ - y₁) / (x₂ - x₁)

= (4 - 0) / (1 - 5)

= 4 / -4

= -1

3 = -1(0) + b

b = 3

Jadi persamaan grafik yang melalui titik (5, 0), (1, 4), dan (0, 3) adalah y = -x + 3.

Penjelasan dengan langkah-langkah:

Untuk menentukan persamaan grafik yang melalui titik (5, 0), (1, 4), dan (0, 3), kita dapat menggunakan metode interpolasi linier.

Misalkan persamaan grafik tersebut adalah y = mx + b, di mana m adalah kemiringan dan b adalah perpotongan dengan sumbu y.

Untuk mencari kemiringan (m), kita dapat menggunakan dua titik pertama dari (5, 0) dan (1, 4):

m = (y₂ - y₁) / (x₂ - x₁)

= (4 - 0) / (1 - 5)

= 4 / -4

= -1

Sekarang kita memiliki kemiringan (m) = -1.

Selanjutnya, kita dapat menggunakan salah satu titik, misalnya (0, 3), untuk mencari b:

3 = -1(0) + b

b = 3

Jadi persamaan grafik yang melalui titik (5, 0), (1, 4), dan (0, 3) adalah y = -x + 3.

Pembahasan

Dari soal diketahui bahwa salah satu akar persamaan kuadrat adalah - 5, yang berarti nilai x adalah 5. Serta akar persamaan yang lain adalah m, dengan substitusi maka nilai m atau akar persamaan yang lain adalah:

x² + 9x + 2m = 0

( - 5 )² + 9( - 5 ) + 2m = 0

25 + ( - 45 ) + 2m = 0

- 20 + 2m = 0

2m = 0 + 20

2m = 20

m = 20/2

m = 10

Nilai diskriminannya 9, karena D > 0 maka persamaan memiliki akar berbeda.Nilai diskriminannya -23, karena D < 0 maka persamaan tidak memiliki akar real.Nilai diskriminannya 0, karena D = 0 maka persamaan memiliki akar yang kembar (identik).

Latihan No. 2

Nilai m adalah 6 atau -2

Langkah Penyelesaian

Dari gambar soal, dapat disimpulkan

D = b² - 4ac

Jika D > 0 maka akar real dan punya penyelesaian yg berbeda.Jika D = 0 maka akar real dan kembar (sama)Jika D < 0 maka akar tidak real

__________________________________

LATIHAN No. 1:

a.) x² + 3x = 0

maka a = 1, b = 3, c = 0

D = 3² - 4(1)(0)

D = 9 - 0

D = 9

Jadi, nilai diskriminannya 9, karena D > 0 maka persamaan memiliki akar berbeda.

b.) 3x² + x + 2 = 0

maka a = 3, b = 1, c = 2

D = 1² - 4(3)(2)

D = 1 - 24

D = -23

Jadi, nilai diskriminannya -23, karena D < 0 maka persamaan tidak memiliki akar real.

c.) 3x² - 6x + 3 = 0

maka a = 3, b = -6, c = 3

D = (-6)² - 4(3)(3)

D = 36 - 36

D = 0

Jadi, nilai diskriminannya 0, karena D = 0 maka persamaan memiliki akar yang kembar (identik).

LATIHAN No. 2:

Ingat, bahwa jika suatu persamaan kuadrat memiliki akar kembar maka diskriminannya sama dengan 0 (D = 0)

x² + mx + (m + 3) = 0

maka a = 1, b = m, c = (m + 3)

D = m² - 4(1)(m + 3)

0 = m² - 4(m + 3)

0 = m² - 4m - 12 (balik, biar lebih mudah)

m² - 4m - 12 = 0

(m - 6)(m + 2) = 0

m - 6 = 0

m1 = 6

m + 2 = 0

m2 = -2

Maka, nilai dari m adalah 6 atau -2

27. Latihan Soal menyusun Persamaan kuadrat baru 1.tentukan Persamaan kuadrat yg ararnya adalah 3 dan -7 2. tenturan Persamaan kuadrat yg akar-akarnya adalah a dan B. Jawaban:

1. Untuk menentukan persamaan kuadrat yang memiliki akar 3 dan -7, kita dapat menggunakan rumus hubungan akar:

Jika u dan B adalah akar-akar persamaan kuadrat, maka persamaan kuadrat dapat ditulis sebagai:

(x - u)(x - B) = 0

Dalam hal ini, u = 3 dan B = -7:

(x - 3)(x - (-7)) = 0

(x - 3)(x + 7) = 0

Persamaan kuadrat baru yang memiliki akar 3 dan -7 adalah:

(x - 3)(x + 7) = 0

2. Untuk menentukan persamaan kuadrat yang memiliki akar-akarnya a dan B, serta hasil kali akar-akarnya adalah -1 dan -20, kita dapat menggunakan rumus hubungan akar:

Jika u dan B adalah akar-akar persamaan kuadrat, maka persamaan kuadrat dapat ditulis sebagai:

(x - u)(x - B) = 0

Jika hasil kali akar-akarnya adalah -1 dan -20, maka kita dapat menulis:

a * B = -1

a * B = -20

Dari dua persamaan di atas, kita bisa menyimpulkan bahwa a = 1 dan B = -20.

Persamaan kuadrat baru yang memiliki akar-akar a dan B adalah:

(x - 1)(x - (-20)) = 0

(x - 1)(x + 20) = 0

3. Untuk menyusun persamaan kuadrat yang memiliki akar 3 dan akar-akar persamaan kuadrat 2x² + 5x - 3 = 0, kita perlu menggunakan hubungan akar dan koefisien:

Diketahui akar persamaan kuadrat pertama (2x² + 5x - 3 = 0) adalah 3.

Jadi, kita akan memiliki persamaan (x - 3)(x - B) = 0.

Untuk menentukan B, kita bisa menggunakan hubungan koefisien kuadratik dan akar. Dalam persamaan kuadrat pertama, produk akar adalah -3/2, dan kita tahu salah satu akarnya adalah 3. Jadi kita bisa tulis:

3 * B = -3/2

B = -1/2

Maka persamaan kuadrat baru adalah:

(x - 3)(x - (-1/2)) = 0

(x - 3)(x + 1/2) = 0

4. Diketahui akar-akar persamaan kuadrat x² + 2x + 3 = 0 adalah a dan B. Untuk menentukan persamaan kuadrat baru yang memiliki akar-akar (a-2) dan (B-2), kita dapat menggunakan rumus hubungan akar:

Jika u dan B adalah akar-akar persamaan kuadrat, maka persamaan kuadrat dapat ditulis sebagai:

(x - u)(x - B) = 0

Jika akar-akarnya adalah (a-2) dan (B-2), maka persamaan kuadrat baru dapat ditulis sebagai:

(x - (a-2))(x - (B-2)) = 0

Menggunakan sifat aljabar, kita bisa menyederhanakan persamaan ini menjadi:

(x - a +2)(x - B +2) = 0

Persamaan kuadrat baru yang memiliki akar-akar (a-2) dan (B-2) adalah:

(x - a +2)(x - B +2) = 0
