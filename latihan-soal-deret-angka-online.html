Pengertian Algoritma

Algoritma adalah susunan logis dan sistematis dari langkah-langkah untuk memecahkan atau menyelesaikan suatu permasalahan. Misalnya adalah langkah – langkah untuk memasak mie instan, langkah – langkah yang dapat kita temukan di balik bungkus mie instan tersebut juga dapat kita sebut sebagai sebuah algoritma karena merupakan langkah-langkah untuk menyelesaikan permasalahan yaitu memasak mi instan menjadi matang.  

Dalam dunia komputer, algoritma merupakan alur pemrosesan pada sistem komputer yang terbentuk dari kumpulan command atau perintah yang terintegrasi pada setiap aspek komputer (brainware, software, dan hardware).  

Dalam dunia pemrograman, algoritma digunakan untuk membuat softwre dengan berbagai kegunaan yang digunakan pada perangkat komputer. Algoritma dalam pembuatan software diimplementasikan dengan beberapa bahasa pemrograman seperti C/C++, C#, Visual Basic, dan bahasa pemrograman lainnya.Tanpa algoritma yang tepat, baris demi baris kode tidak akan menghasilkan output yang diinginkan. Oleh karena itu algortima sangat penting dalam membuat program berfungsi sebagaimana mestinya.

Pembahasan

nilai N dari deret angka 3,6,9,12.....N. dimana N adalah nilai deret ke sekian kita perlu pelajari pola dari deret tersebut.

Kita satukan persepsi bahwa indeks deret tersebut dimulai dari N = 0.

Jika deret adalah 3,6,9,12.....N, kita bisa tuliskan:

N = 0, value = 3

N = 1, value = 6

N = 2, value = 9

N = 3, value = 12

Dari statement tersebut, value dapat kita rumuskan dengan persamaan N berikut ini

value = (N + 1) x 3

Kesimpulan

Maka algoritmanya adalah sebagai berikut:

MulaiInput nilai NValue = (N + 1) x 3Cetak ValueSelesai

#include<stdio.h>

#include<conio.h>

#include<string.h>  

int main()

{

int N;

printf("Input nilai N : ");

scanf("%i", &N);

int value = (N + 1) * 3;

printf("Value pada deret ke N : %i", value);

return 0;

}

29. Soal deret angka 94 92 46 44 22 20 10 ?


Jawab

8

Penjelasan dan Langkah Langkah

Untuk menemukan pola dalam deret angka yang diberikan, perhatikan pola pengurangan antara angka-angka tersebut:

94 - 2 = 92

92 : 2 = 46

46 - 2 = 44

44 : 2 = 22

22 - 2 = 20

20 : 2 = 10

10 - 2 = 8

Dalam pola tersebut, setiap angka pertama dikurangi dengan angka 2, dan hasil pengurangan tersebut menjadi angka kedua pada pasangan angka berikutnya. Selanjutnya, angka kedua dibagi dengan angka 2 , dan hasilnya menjadi angka ketiga pada pasangan angka berikutnya. Pola pengurangan ini berlanjut hingga angka terakhir dalam deret.

Jika kita teruskan pola pengurangan ini, maka angka berikutnya akan menjadi:

10 - 2 = 8

Jadi, angka berikutnya dalam deret adalah 8.
