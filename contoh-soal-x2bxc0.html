16. Persamaan kuadrat ax²+bx+c=0, mempunyai akar x1 dan x2. Bila x1+x2=3 dan x1.x2=-1/2, persamaan kuadrat tersebut adalah


Persamaan kuadrat tersebut adalah 2x² - 6x - 1 = 0.

Pendahuluan

Persamaan kuadrat adalah persamaan dengan variabel berpangkat maksimal 2 dengan bentuk umum persamaan kuadrat ax² + bx + c = 0.

Rumus persamaan kuadrat baru

(x - x₁) (x - x₂) = 0  atau x² - (x₁ + x₂) x + x₁.x₂ = 0

Pembahasan

Diketahui :

Persamaan kuadrat ax² + bx + c = 0, mempunyai akar x₁ dan x₂.

x₁ + x₂ = 3

x₁.x₂ = [tex]\displaystyle -\frac{1}{2}[/tex]

Ditanya :

Persamaan kuadrat tersebut.

Penyelesaian :

Menentukan persamaan kuadrat

Masukkan nilai x₁ + x₂ = 3 dan x₁.x₂ = [tex]\displaystyle -\frac{1}{2}[/tex] ke dalam rumus persamaan kuadrat.

x² - (x₁ + x₂) x + x₁.x₂ = 0

x² - 3x - [tex]\displaystyle\frac{1}{2}[/tex] = 0       (kesemua ruas dikali 2, untuk menghilangkan pecahan)

2x² - 6x - 1 = 0

Jadi persamaan kuadrat tersebut adalah 2x² - 6x - 1 = 0.

19. persamaan kuadrat dari cx²×bx×+a=0,c≠0 mempunyai akar akar x1 dan x2,maka nilai x1 +x2 adalah​


Nilai x1 + x2 adalah 0. Soal di atas adalah soal tentang persamaan kuadrat.

Penjelasan dengan langkah-langkah:

Diketahui:

Persamaan kuadrat cx² + bx + a = 0, dengan c ≠ 0, memiliki akar-akar x1 dan x2.

Ditanya:

Nilai dari x1 + x2.

Jawab:

Untuk menemukan nilai x1 + x2, kita dapat menggunakan fakta bahwa jumlah akar-akar persamaan kuadrat adalah negatif koefisien dari x² dibagi dengan koefisien x.

Untuk mencari nilai dari x1 + x2 dari persamaan kuadrat cx² + bx + a = 0, kita perlu mengingat rumus untuk menjumlahkan akar-akar persamaan kuadrat.

Dalam persamaan kuadrat umum ax² + bx + c = 0, akar-akarnya diberikan oleh rumus:

x1 = (-b + √(b² - 4ac)) / 2a

x2 = (-b - √(b² - 4ac)) / 2a

Dalam kasus ini, kita memiliki persamaan cx² + bx + a = 0. Dengan membandingkan koefisien, kita memiliki a = c dan b = 0.

Menggunakan rumus tersebut, kita bisa menemukan akar-akar persamaan ini:

x1 = (0 + √(0 - 4ac)) / 2c

x2 = (0 - √(0 - 4ac)) / 2c

Namun, perlu diingat bahwa nilai c ≠ 0. Jadi, kita tidak bisa langsung menggunakan rumus di atas karena akan menghasilkan pembagian dengan nol.

Untuk menemukan nilai x1 + x2, kita bisa menggunakan fakta bahwa jumlah akar-akar persamaan kuadrat adalah negatif koefisien dari x² dibagi dengan koefisien x.

Jadi, x1 + x2 = - (b/a)

Dalam kasus kita, b = 0 dan a = c, jadi kita bisa menuliskan:

x1 + x2 = - (0/c) = 0

Jadi, nilai x1 + x2 adalah 0.

38. Persamaan kuadrat dari cx²×bx× a=0,c≠0 mempunyai akar akar x1 dan x2,maka nilai x1 x2 adalah​


Untuk persamaan kuadrat dalam bentuk umum ax^2 + bx + c = 0, dengan c ≠ 0, akar-akar x1 dan x2 dapat dihitung menggunakan rumus kuadrat:

x1 = (-b + √(b^2 - 4ac)) / 2a

x2 = (-b - √(b^2 - 4ac)) / 2a

Kita telah diberikan persamaan cx^2 + bx + a = 0, tetapi karena rumus kuadrat menggunakan koefisien a, b, dan c, kita perlu menukar posisi a, b, dan c dari persamaan yang diberikan agar sesuai dengan bentuk umum ax^2 + bx + c = 0.

Sehingga persamaan yang sesuai untuk dihitung adalah ax^2 + bx + c = 0, dengan a ≠ 0, dan kita memiliki a = c, b = b, dan c = a.

Dengan mengganti a, b, dan c pada rumus kuadrat, maka nilai x1 dan x2 dari persamaan cx^2 + bx + a = 0 adalah:

x1 = (-b + √(b^2 - 4ac)) / 2a = (-b + √(b^2 - 4ca)) / 2c

x2 = (-b - √(b^2 - 4ac)) / 2a = (-b - √(b^2 - 4ca)) / 2c

Maka nilai x1 * x2 adalah:

x1 * x2 = [(-b + √(b^2 - 4ca)) / 2c] * [(-b - √(b^2 - 4ca)) / 2c]

x1 * x2 = [(-b)^2 - (b^2 - 4ca)] / (4c^2)

x1 * x2 = (4ca) / (4c^2)

x1 * x2 = a / c

Jadi, nilai x1 * x2 dari persamaan cx^2 + bx + a = 0 adalah a / c.
