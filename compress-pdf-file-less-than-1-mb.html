Jika erza ingin mengisi file sekolah sebesar 875 MB dan harus menghapus beberapa file musik agar kapasitas cukup, maka banyak file musik yang tersisa adalah.... MB

(Musik 20% dari 4000MB)

Banyak file musik yang terisa adalah 0 MB dan file sekolah masih membutuhkan 75 MB kapasitas tambahan.

21. Resolusi sebuah file gambar adalah 2048 x 2048 dengan kedalaman 16bit, berapakah size file gambar tersebut?A. 8 MBB. 7 MBC. 9 MBD. 6 MB


Mapel : TI
Materi : Pengeditan Gambar
Sub Materi : Resolusi Gambar
————————————————————
Maka Ukuran File Asli ....
= 2048 × 2048 × 16 / 8
= 8.388.608 byte

Maka ubah ke Kilobyte ( KB ) ....
= 8.388.608 / 1.024
= 8.192 KB ...

Terakhir, Ubah ke Megabyte ( MB ) ....
= 8.192 / 1.024
= 8 MB ...... ✔️

32. 5. Brian memiliki memori HP dengan kapasitas 1 GB (1 GB = 1024 MB), Mermori tersebut berisi file-file berikut : 1. foto = 250 MB 2. Aplikasi = 425 MB 3. Dokumen = 50 MB 4. Audio = 200 MB. Brian ingin menyimpan file video dengan ukuran 300 MB. Untuk itu Brian harus menghapus salah satu file yang sudah ada. Brian hanya ingin menghapus file yang berukuran sekecil-kecilnya. Tentukan file mana yang harus dihapus Brian agar dapat menyimpan file video tersebut?​


Mari kita hitung total penggunaan memori saat ini:

Total penggunaan saat ini = 250 MB (foto) + 425 MB (aplikasi) + 50 MB (dokumen) + 200 MB (audio) = 925 MB

Kapasitas memori yang tersedia = 1024 MB (1 GB) - 925 MB = 99 MB

Brian ingin menyimpan file video dengan ukuran 300 MB, namun hanya memiliki sisa 99 MB. Oleh karena itu, Brian harus menghapus salah satu file yang sudah ada.

Dari semua file yang ada, file dengan ukuran paling kecil adalah "Dokumen" yang berukuran 50 MB. Jadi, Brian harus menghapus file "Dokumen" agar dapat menyimpan file video berukuran 300 MB.
