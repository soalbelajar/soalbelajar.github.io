no 1. uang vina = 51000
membeli buku 4/8 dari 51000
dit: sisanya?
jwb: 4/8 x 51000 = 25500
jadi sisa uang = 51000 - 25500 = 25500 (sisa uang)
no 2. 56 buku bacaan
2/4 dibaca kelas VI
dit: buku yg tdk dibaca?
2/4 x 56 = 28 (buku yg dibaca kls VI)
yg tdk dibaca = 56 - 28 = 28 buku
no 3. 4/6 x 21 = 14
21 - 14 = 7 pohon yg belum berbuah
no 4. 3/9 x 300m² = 100m²
no 5. 6/8 x 160 = 120halaman
no 6. usia ibu = 5/6 x 36 = 30th
selisih = 36 - 30 = 6th
no 7. laki-laki = 4/8 x 440 = 220
perempuan = 440 - 220 = 220
no 8. 3/12 x 340.000 = 85.000
sisa = 340.000 - 85.000 = 255.000
no 9. 4/6 x 18 = 12butir
