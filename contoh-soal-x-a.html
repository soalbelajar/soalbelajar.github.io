6. Contoh Soal Ujian Sekolah SMP Pemfaktoran bentuk x2 + 2x – 48 adalah ... A. (x - 6)(x - 8) B. (x + 8)(x – 6) C. (x – 4)(x – 12) D. (x + 24)(x – 2)


Persamaan kuadrat adalah suatu bentuk persamaan yang memiliki pangkat tertinggi dari variabelnya dua. Berdasarkan soal, dapat disimpulkan bahwa pemfaktoran bentuk x² + 2x – 48 adalah (x + 8)(x – 6) (B).

Penjelasan dengan langkah-langkah:

Diketahui:
Persamaan kuadrat x² + 2x – 48.

Ditanyakan:

Tentukan pemfaktoran bentuk x² + 2x – 48!

Jawab:

Ketika kita mempelajari mata pelajaran Matematika, maka kita akan belajar tentang pemfaktoran pada persamaan kuadrat. Pemfaktoran bentuk x² + 2x – 48 yaitu:
[tex]x^{2} + 2x-48=(x+8)(x-6).[/tex]

Dengan demikian, pemfaktoran bentuk x² + 2x – 48 adalah (x + 8)(x – 6) (B).

24. Contoh Soal Ujian Sekolah SD Hasil -1.836 : ( -9 ) x ( -14 ) = …. A. -326 B. -336 C. -2.846 D. -2.856


Hasil -1.836 : ( -9 ) x ( -14 ) adalah -2.856.

Penjumlahan

a) (+) + (+) = (+)

Contoh: 3 + 9 = 12

b) (-) + (-) = (-)

Contoh: ( -2) + (-3) = (-5)

c) (+) + (-) = bisa (+) bisa juga (-)

Jika (+) lebih besar maka hasilnya (+)

Jika (-) lbh besar maka hasilnya (-)

Contoh:

9 + (-6) = 3 → hasil positif

9 + (-16) = (-7) → hasil negatif

d) (-) + (+) = bisa (+) bisa juga (-)

Jika (+) lebih besar maka hasilnya (+)

Jika (-) lebih besar maka hasilnya (-)

Contoh:

(-5) + 2 = (-3) → hasilnya negatif

(-5) + 9 = 4 → hasilnya positif

Pengurangan

a) (+) - (+) = bisa (+) bisa juga (-)

Apabila bilangan pertama lebih besar maka hasilnya positif

Apabila bilangan kedua lebih besar maka hasilnya negatif

Contoh:

5 - 2 = 3 → hasil positif

5 - 9 = (-4) → hasil negatif

b) (-) - (-) = bisa (+) bisa juga (-)

Apabila bilangan pertama lebih besar dari bilangan kedua maka hasilnya (+)

37. Contoh Soal UTBK SNPMB Himpunan penyelesaikan dari pertidaksamaan | x – 1| < 3 – |x| dengan x adalah bilangan R adalah.. A. – 2 < x < -1 B. -1 < x < 2 C. x < -1 atau x > 2 D. x < -2 atau x > 1 E. 1 < x < 2


Jika diberikan pertidaksamaan [tex] \rm |x-1| <3-|x|[/tex] dengan x adalah bilangan R. Maka himpunan penyelesaiannya adalah [tex] \rm -1\leqslant x <2[/tex] (opsi B).

Penjelasan dengan langkah-langkah

Diketahui :

[tex] \rm |x-1| <3-|x|[/tex] dengan x adalah bilangan R

Ditanya :

HP?

Jawab :

Dari pertidaksamaan terdapat dua tanda mutlak yaitu |x-1| dan |x|.

[tex] |x-1| = \begin{cases}x-1, ~~~ untuk\: x-1 \geqslant 0 \\ -x+1, ~~~ untuk \:x -1<0 \end{cases} [/tex]

[tex] |x| = \begin{cases}x, ~~~ untuk\: x \geqslant 0 \\ -x, ~~~ untuk \:x < 0 \end{cases} [/tex]

Terdapat 4 kemungkinan :

●Kemungkinan 1 : |x-1| positif dan |x| positif

[tex] \rm |x-1| <3-|x|[/tex]

[tex] \rm x-1<3-x[/tex]

[tex] \rm x+x<3+1[/tex]

[tex] \rm 2x<4[/tex]

[tex] \rm x <4\div 2[/tex]

[tex] \rm x <2[/tex]

|x-1| syaratnya [tex] \rm x-1 \geqslant 0 \rightarrow x \geqslant 1 [/tex]

|x| syaratnya [tex] \rm x \geqslant 0[/tex]

Irisan ketiganya : [tex] \rm 1\leqslant x <2[/tex] ...(1)

[tex] \\[/tex]

●Kemungkinan 2 : |x-1| positif dan |x| negatif

[tex] \rm |x-1| <3-(-|x|)[/tex]

[tex] \rm x-1<3+x[/tex]

[tex] \rm x-x<3+1[/tex]

[tex] \rm 0<4[/tex] ... (penyelesaian tidak mungkin)

|x-1| syaratnya [tex] \rm x-1 \geqslant 0 \rightarrow x \geqslant 1 [/tex]

-|x| syaratnya [tex] \rm x < 0[/tex]

Irisan keduanya tidak ditemukan.

[tex] \\[/tex]

●Kemungkinan 3 : |x-1| negatif dan |x| positif

[tex] \rm -|x-1| <3-|x|[/tex]

[tex] \rm -x+1<3-x[/tex]

[tex] \rm -x+x<3-1[/tex]

[tex] \rm 0<2[/tex] ... (penyelesaian tidak mungkin)

|x-1| syaratnya [tex] \rm x-1 < 0 \rightarrow x < 1 [/tex]

|x| syaratnya [tex] \rm x \geqslant 0[/tex]

Irisan keduanya :[tex] \rm 0\geqslant x <1[/tex] ...(2)

[tex] \\[/tex]

●Kemungkinan 4 : |x-1| negatif dan |x| negatif

[tex] \rm -|x-1| <3-(-|x|)[/tex]

[tex] \rm -x+1<3+x[/tex]

[tex] \rm -x-x<3-1[/tex]

[tex] \rm -2x<2[/tex]

[tex] \rm x >2\div (-2)[/tex]

[tex] \rm x >-1[/tex]

-|x-1| syaratnya [tex] \rm x -1 <0 \rightarrow x<1[/tex]

-|x| syaratnya [tex] \rm x<0[/tex]

Irisan gabungan ketiganya : [tex] \rm -1<x<0[/tex] ...(3)

[tex] \\[/tex]

Gabungkan pertidaksamaan (1), (2), dan (3) tadi, maka diperoleh HP : [tex] \rm -1\leqslant x <2[/tex].
