1.PDB- Palm File Database. Dapat menyertakan beberapa format buku digital yang berbeda, yang ditujukan untuk perangkat berbasiskan sistem operasi Palm. Pada umumnya digunakan untuk buku digital berformat PalmDOC (AportisDoc) dan format eReader juga.

2.PDF- Portable Document Format yang diciptakan oleh Adobe untuk produk Acrobat mereka. Format ini secara tidak langsung merupakan format yang digunakan untuk pertukaran dokumen. Dukungan perangkat lunak untuk format ini hampir mencakupi semua platform komputer dan perangkat genggam.

Beberapa perangkat memiliki masalah dengan PDF karena kebanyakan konten yang tersedia akan ditampilkan baik untuk format A4 atau surat, yang keduanya tidak mudah dibaca ketika diperkecil sesuai layar kecil. Beberapa aplikasi pembaca buku digital dapat menyusun ulang tampilan beberapa dokumen PDF, termasuk Sony PRS505, untuk mengakomodasi layar kecil.

baca juga cara memperkecil ukuran file PDF

3.HTML- Hyper Text Markup Language adalah tulang punggung dari World Wide Web. Banyak teks yang didistribusikan dalam format ini. Selain itu, beberapa pembaca e-book mendukung Cascading Style Sheets (CSS) yang pada dasarnya gayautama panduan untuk halaman HTML.

4 CHM- Compressed HTML, sering digunakan untuk file bantuan Windows. Hal ini telah menjadi sangat populer untuk distribusi teks dan bahan pendukung lainnya melalui Web.

31. sebutkan kelemahan dan kelebijan sistem operasi pada : windows 95 windows 98 windows xp windows vista windows 7 windows 8


kelemahan:
Sistem Operasi Windows 3.1.1
Kelebihan dari sistem operasi ini adalah :
a. Memudahkan komunikasi antar anggota workgroup lainnya
b. Menggunakan sharing printer
c. Memudahkan mengakses file dan program aplikasi yang berada di komputer lain.
d. Administrasi sistem jaringan (network) relative lebih mudah

Sistem Operasi Windows 95
Keunggulan dari sistem operasi ini adalah :
a. Sangat mudah dan cepat :
· Setiap saat kita dapat menggunakan seluruh kemampuan Windows 95, yaitu dengan mengklik tombol start untuk menjalankan program aplikasi, membuka dokumen, mencari data dan menggunakan fasilitas lainnya.
· Pada taskbar Windows 95 ditampilkan seluruh program aplikasi yang sedang anda gunakan, hal ini akan memudahkan untuk berpindah program aplikasi.
b. Berbagai fasilitas baru
1. Dengan sistem 32-bit, Kemampuan pengkodeaan 32-bit dari prosesor seri 80386, 80486, dan 80586 dapat dimanfaatkan secara optimal.
2. Dilengkapi fasilitas multithreading dan preemptive multitasking : multitasking lebih responsive dan dengan fasilitas ini dapat menjalankan beberapa program aplikasi secara bersama-sama.
3. Kemampuan memberi nama yang panjang pada file dan folder (maksimal 250 karakter).
4. Lalu lintas input/output yang lebih cepat yang membuat operasi baca tulis, dan mencetak ke printer menjadi lebih ditingkatkan kecepatannya.
5. Visual device driver 32-bit yang mengelola sumber daya sistem sehingga memungkinkan lebih dari satu aplikasi dapat mengunakan daya pada saat yang bersamaan.
Securiti secara terpusat : Sistem pengamanan dapat dilakukan secara terpusat pada server.
c. Fasilitas Bantuan (Help) yang terintegrasi Fasilitas Help apabila pemakai sedang menghadapi kesulitan dalam pengoprasian, program ini lebih luas.

Sistem Operasi Windows 98
Keunggulan dari sistem operasi ini dalah :
