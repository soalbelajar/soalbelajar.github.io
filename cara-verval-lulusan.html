24. Diketahui peluang Joko lulus ulangan matematika 0,75 dan peluang Tini lulus ulangan matematika 0,6 Tentukan peluang :- keduanya lulus ulangan matematika- joko lulus dan tini tidak luluspake cara ya kak


Peluang joko lulus = 0,75
peluang Tini lulus = 0,6

peluang joko tidak lulus=1-0,75 = 0,25
peluang tini tidak lulus= 1-0,6= 0,4

jika peluang keduanya lulus maka
peluang joko lulus x peluang tini lulus
0,75x0,6 = 0,45

jika peluang joko lulus dan tini tidak lulus
maka peluang joko lulus x peluang tini tidak lulus

0,75 x 0,4 =0,3

33. Peluang andika lulus matematika adalah 0,85 peluang roni lulus matematika 0,80 maka peluang salah satu dari mereka tidak lulus matematika adalah... #pakai cara plis


Pa = peluang andika lulus = 0.85
P'a = peluang andika tidak lulus = 1- 0.85 = 0.15
Pr = peluang roni lulus=0.8
P'r = peluang roni tidak lulus= 1-0.8 = 0.2

Peluang andi lulus roni tidak lulus atau andi tidak lulus roni lulus
Pa × P'r + P'a × Pr
0.85 × 0.2 + 0.15×0.8
0.17 + 0.12
0.29.
