Pengertian fungsi string pada excel

fungsi string adalah fungsi yang di gunakan untuk mengambil karakter sebanyak n pada posisi tertentu. fungsi string sendiri terbagi menjadi tiga bagian yaitu fungsi left, fungsi right dan fungsi mid. secara bahasa indonesia artinya left (kiri) mid (tengah) dan right (kanan) yang akan kami bahas satu persatu secara gamblang di bawah ini:
Fungsi left, fungsi left ini di gunakan untuk mengambil karakter dari sebelah kiri contoh : = left (x,z)
fungsi mid, fungsi mid ini di gunakan untuk mengambil karakter dari tengah contoh : = mid (x,y,z)
fungsi right, fungsi right ini di gunakan untuk mengambil karakter dari sebelah kanan contoh : = right (x,z).
Keterangan rumus:
x : cell kunci atau cell yang akan di ambil karakternya
y : jumlah karakter
n : jumlah karakter yang di ambil

Pengertian dari String sendiri adalah gabungan dari beberapa karakter pada sebua kata.

Ada beberapa jenis operator string. Jenis string pertama adalah berupa gabungan dari oeprator yang kemudian gabungan tersebut berasal dari kanan dan kiri. ?>

contoh fungsi ord():

?>

3. md5()
md5() memiliki fungsi untuk melakukan enkripsi pada salah satu string dengan menggunakan enkripsi md5. Pada penggunaan metode string md5() paling sering digunakan untuk melakukan enkripsi pada password.

contoh fungsi md5():
<?php
$str = "Nama Awal: Modul \nNama Akhir: Makalah";

?>

19. 2. Jelaskan dari fungsi string left, right dan mid ( ( Microsoft Excel ) ?​


1. Fungsi LEFT

Fungsi LEFT digunakan untuk mengambil sejumlah karakter tertentu suatu teks yang dimulai dari karakter pertama atau karakter paling kiri sampai sejumlah karakter tertentu sesuai dengan yang kita inginkan.

2. Fungsi RIGHT

Kebalikan dari fungsi LEFT, fungsi RIGHT digunakan untuk mengambil sejumlah karakter tertentu suatu teks dari karakter terakhir sampai dengan sejumlah karakter tertentu yang diinginkan.

3. Fungsi MID

Fungsi MID mengambil sejumlah karakter dari karakter kesekian sampai dengan sejumlah karakter yang diinginkan. Fungsi MID bergerak dari kiri ke kanan sama dengan fungsi LEFT, hanya saja bukan dari karakter pertama.

24. Contoh Soal IF tunggal (minimal 4 soal ) Contoh Soal IF (string,left,right,mid) (minimal 4 soal) Contoh Soal IF AND (minimal 4 soal) Contoh Soal IF OR ( minimal 4 soal)


Misalkan kita ingin menampilkan keterangan"Lulus" atau "Tidak Lulus" pada kolom keterangan berdasarkan rumus :

Total = ( Nilai Bhs. Ind. + Nilai Mat ) / 2.

Siswa dinyatakan lulus jika Total >= 60. Jadi jika Total di bawah 60 berarti tidak lulus.

Formula Excel yang ditulis pada sel F6 adalah :
=IF( SUM(D6:E6)/2 >= 60 ; "Lulus" ; "Tidak Lulus" )
Jika syarat kelulusan diubah, misalnya : siswa dinyatakan lulus jika nilai pelajaran Bhs. Indonesia >= 70 dan nilai Matematika >= 60. Maka formula yang harus ditulis adalah :
=IF( AND( D6 >= 70 ; E6 >= 60 ) ; "Lulus" ; "Tidak Lulus" )
Sedangkan untuk menghitung banyaknya siswa yang lulus dan tidak lulus, dapat memakai fungsi COUNTIF() sebagai berikut :
=COUNTIF( $F$6:$F$15 ; "Lulus" )
dan
=COUNTIF ( $F$6:$F$15 ; "Tidak Lulus" )
Note : tanda ";" (titik koma) dapat diganti dengan "," (koma)

FUNGSI AND
Menghasilkan nilai BENAR jika semua ketentuan yang diuji bernilai benar dan SALAH jika ada satu atau lebih ketentuan yang bernilai salah.

1. Fungsi left, fungsi left ini di gunakan untuk mengambil karakter dari sebelah kiri contoh : = left (x,z)
2. fungsi mid, fungsi mid ini di gunakan untuk mengambil karakter dari tengah contoh : = mid (x,y,z)
3. fungsi right, fungsi right ini di gunakan untuk mengambil karakter dari sebelah kanan contoh : = right (x,z).
Keterangan rumus:
1. x : cell kunci atau cell yang akan di ambil karakternya
2. y : jumlah karakter
3. n : jumlah karakter yang di ambil

39. sebutkan dan jelaskan fungsi string dan fungsi statistik pada MS Excel


fungsi string sama seperti fungsi teks. 
tujuannya untuk memudahkan kita mengambil beberapa karakter yang berposisikan di kiri, tengah, atau kanan dari teks.
ada 3 fungsi, right, mid, left, upper, lower, proper
- LEFT
fungsi left digunakan untuk mengambil karakter dari kiri pada string/teks.
=LEFT(kata,x) atau =LEFT(alamatsel,x)
contohnya kata KETERAMPILAN, di sel A2 misalnya.
=LEFT(A2,4) --> yang muncul adalah KETE

-RIGHT
sama kaya left, 
=RIGHT(kata,x) atau =RIGHT(alamatsel,x)
contoh : =RIGHT(A2,5) --> PILAN (dimulai dari 5 karakter dari ujung kanan)

-MID 
kalo ini buat ditengah2
=MID(kata,dimulaidarikarakterke-n,n huruf)
contoh : MID(A2,3,4) --> TERA (T = karakter ke 3 dari kiri) (tera soalnya 4 karakter yang diambil)

- UPPER buat ngubah bbrp karakter jadi huruf kapital,
=UPPER("daftar") --> DAFTAR

-LOWER buat ngubah bbrp karakter jadi huruf kecil
=LOWER("DAFTAR") --> daftar

-PROPER buat ngubah jadi besar dan kecil kaya huruf alay

fungsi statistik fungsi yang digunakan untuk mngolah data yang berupa numerik hanya saja operasi ini dilakukan dengan statistik
terdiri dari fungsi MIN,MAX,AVERAGE,dan COUNT
-MIN untuk menampilkan nilai numerik yang terkecil
-MAX untuk menampilkan nilai numerik yang terbesar
-AVERAGE untuk menampilkan rata2
-COUNT untuk menghitung jumlah sel dalam suatu range atau grup data yang ada nilainya
