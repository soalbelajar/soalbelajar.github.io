8. Diketahui fungsi f, g, dan h pada bilangan real dan definisikan f(x)=x², g(x)=5x-3 , dan h(x)=√x+1. Tentukan komposisi fungsi berikut ini. A. (f o g) (x) B. (g o f o h) (x)


Diketahui fungsi f, g, dan h pada bilangan real dan definisikan f(x)=x², g(x)=5x-3 , dan h(x)=√x+1. Tentukan komposisi fungsi berikut ini.

A. (f o g) (x)
B. (g o f o h) (x) adalah .....

⇒(f o g)(x) = 25x² - 30x + 9 ✔️

⇒(g o f o h)(x) = 5x + 2 ✔️

____________________________________

◾ Bentuk-bentuk Komposisi Fungsi :

1. (f o g) (x) = f(g(x))

2. (g o f) (x) = g(f(x))

3. f o (g o h) = (f o g) o h
atau f(g(h(x))).

_____________________________________

◾ Penyelesaian.

• Dik :

f(x) = x²
g(x) = 5x - 3
h(x) = √x + 1

• Dit :

(f o g) (x) = ....... ?
(g o f o h) (x) = ....... ?

• Jawaban :

①] (f o g) (x)

(f o g)(x) = f(g(x))

(f o g)(x) = f (5x - 3)

(f o g)(x) = (5x - 3)²

(f o g)(x) = (5x - 3)(5x - 3)

(f o g)(x) = 25x² - 30x + 9 ✔️

=======================

②] (g o f o h) (x)

(g o f o h)(x) = g(f(h(x)))

(g o f o h)(x) = g ( f (√x + 1))

(g o f o h)(x) = g ((√x + 1)²)

Karena dikuadratkan, maka akarnya habis.

(g o f o h)(x) = g (x + 1)

(g o f o h)(x) = 5(x + 1) - 3

(g o f o h)(x) = 5x + 2 ✔️

____________________

◾ Jadi, hasil dari (f o g)(x) = 25x² - 30x + 9 dan (g o f o h)(x) = 5x + 2.
