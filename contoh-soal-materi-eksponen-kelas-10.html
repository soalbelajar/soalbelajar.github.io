6. Materi EKSPONENkelas 10 SMAtolong bantu kak​


[tex] \frac{ {2}^{n+2} - {2}^{n+1}}{ {2}^{n} - {2}^{n+1}} [/tex]

Karena properti dari eksponensial adalah:

[tex] {a}^{n+d} = {a}^{n} \times {a}^{d} [/tex]

[tex] {a}^{n-d} = \frac{ {a}^{n} }{ {a}^{d} } [/tex]

Maka fungsi akan menjadi:

[tex] \frac{ {2}^{n} \times {2}^{2} - {2}^{n} \times 2 }{ {2}^{n} - {2}^{n} \times 2 } [/tex]

Kemudiaan faktorkan dan karena 2^n adalah term yang sama maka hilang / cancel. Lalu 2^2 adalah 4 sehingga 4 - 2 adalah 2:

[tex] \frac{ {2}^{n}({2}^{2} - 2) }{ {2}^{n}(1-2) } = \frac{2}{-1} [/tex]

Maka nilainya adalah:

[tex] \frac{ {2}^{n+2} - {2}^{n+1}}{ {2}^{n} - {2}^{n+1}} = - 2[/tex]

12. Materi Persamaan Eksponen Kelas X


Materi Persamaan Eksponen 
Matematika X

[tex]3^{2x+2}+8.3^{x}-1=0[/tex]
Misalkan 
[tex]p=3^{x}[/tex]
Maka
[tex]9.(3^{x})^{2}+8.3^{x}-1=0[/tex]
Menjadi
[tex]9.p^{2}+8.p-1=0[/tex]
Faktorkan
(9p - 1)(p + 1) = 0
Diperoleh 
[tex]p= \frac{1}{9}[/tex]
Atau
[tex]p=-1[/tex]
Selanjutnya, kembalikan p ke bentuk semula

Bagian Pertama
[tex]3^{x}= \frac{1}{9}[/tex]
[tex]3^{x}= 3^{-2}[/tex]
x = - 2

Bagian Kedua
[tex]3^{x}=-1[/tex]
Tidak ada nilai x yang memenuhi

Mengenai interval, perhatikan dengan cermat penjelasan berikut ini.
⇒ Telah diperoleh penyelesaian yakni x = -2
⇒ Interval yang tertulis di pilihan ganda adalah,
A. -¹/₂ ≤ x ≤ 0
B. -3 ≤ x ≤ -1 
C. -¹/₂ ≤ x ≤ ¹/₂
D. ¹/₂ ≤ x ≤ 1
E. 1 ≤ x ≤ 2

Jadi, posisi x = -2 berada di interval [-3, -1]

--------------------------------
Keterangan:
Tanda interval [a, b] bermakna a ≤ x ≤ b, sedangkan interval (a, b) bermakna a < x < b, jadi kalau [a, b) artinya a ≤ x < b

18. contoh soal eksponen kelas 10bikin sama jawabannya​


Jawaban:

Diketahui akar-akar persamaan dari persamaan eksponensial 32y + 1 – 28 . 3y + 9 = 0 adalah y1 dan y2. Apabila nilai akar y1 > y2, tentukan berapakah nilai 4y1 – y2 …

Pembahasan:

32y + 1 – 28 . 3y + 9 = 0

3 . (3y) – 28 . 3y + 9 = 0

Agar solusi dari persamaan eksponensial bisa didapatkan maka 3^y harus diubah menjadi variabel sendiri misalkan X. Persamaan eksponensial menjadi:

3X² - 28X+9 = 0

(X – 9) (3X – 1) = 0

Sehingga nilai variabel dari persamaan eksponensial adalah X = 9 dan X = 1/3. Akar kedua persamaan adalah:

a) X = 3^y

3² = 3^y

y = 2

b) X = 3^y

1/3 = 3^y

3ˉ¹ = 3^y

y = -1

Maka akar y1 = 2 dan y2 = -1 sehingga nilai 4y1 – y2 bisa dihitung dengan mensubstitusikan akar-akarnya:

4y1 – y2

= 4 (2) - (-1)

= 8 + 1

= 9

- Basis pangkat sama dengan 1
x² + 7x + 10 = 1
x² + 7x + 9 = 0

x = 1/2 (-7 + √13)
x = 1/2 (-7 - √13)

- Fungsi pangkat sama:
3x - 2 = 5x - 4
2x = 2
x = 1

HP = {x | 1/2 (-7 - √13), -5, 1/2 (-7 + √13), -2, 1}

bagian h.

- Basis sama dengan 1

x² - x = 0

x = 0 dan x = 1
