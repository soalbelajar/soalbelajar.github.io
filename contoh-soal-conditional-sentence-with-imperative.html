18. contoh soal conditional sentence type 1 2 3


Contoh soal conditional sentence type 1, 2 dan 3 yaitu

Change the verb in parentheses to first, second, and third conditional.

We (build) _______ a new house, if we (have) _______ enough money.

Answer:

Type 1: We will build a new house, if we have enough money.

Type 2: I would build a new house, if we had enough money

Type 3: I would have bought a new house, if we had had enough money.

39. buat 3 contoh kalimat Imperative Sentence•negative with Adjective dan postive with Ajdective #MAKASIH#JANGAN NGASAL​


Negative + Adjective

Don't be late for school! (jangan terlambat ke sekolah!)Don't be noisy in the classroom! (jangan berisik di dalam kelas!)Don't be naughty to other kids! (jangan nakal ke anak-anak lain!)

Positive + Adjective

Be cautious to your surroundings! (Waspadalah terhadap sekitarmu!)Be humble to others! (Bersikap rendah hatilah kepada orang lain!)Be gone! (Pergi!)

IMPERATIVE SENTENCE

Imperative sentence berarti kalimat perintah. Kalimat ini digunakan ketika ingin memberikan perintah/instruksi kepada orang lain.

Pola Kalimat Imperative

Kalimat imperative biasanya menggunakan kata kerja pertama (Verb 1), seperti:

Get, put, take, bring, close, open, dll

Namun ada juga versi menggunakan be diikuti adjective, seperti:

Be quiet, be wary, be nice, be generous, dll

Untuk kalimat negatifnya cukup gunakan don't di awal kalimat.

Don't open the door.Don't close the window.Don't be mad.Don't be so cruel.
